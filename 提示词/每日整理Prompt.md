# 每日整理 Prompt

这份 prompt 只负责“单次执行”。

长期规则、目录约束、frontmatter 规范、允许与禁止事项，统一以 [LLM维护手册.md](../LLM维护手册.md) 为准。

```text
你现在要执行一次“每日整理”。

在开始前，先读取并遵循：
- LLM维护手册.md

本次输入范围：
- 收集/Inbox.md
- 收集目录下今天新增的原始输入
- 今日日志：日志/YYYY/YYYY-MM-DD.md
- 如果今日日志引用了已有笔记，可以一并读取相关的感悟/知识/展望文件

本次任务目标：
1. 保留原始输入，不删除原始记录；
2. 把今天新增内容整理进更稳定的位置；
3. 优先更新已有笔记，只有在确实是新主题时才新建文件；
4. 输出一次清晰、克制、可复核的整理结果。

请按下面步骤执行：
1. 概括今天原始输入里最重要的 3 到 5 个点；
2. 更新今日日志；
3. 把适合长期保留的个人判断整理到 `感悟/`；
4. 把适合长期执行的目标或行动整理到 `展望/`；
5. 如果今天有明确的新知识点，再整理到 `知识/`；
6. 列出仍然需要我确认的内容。

最后输出：
- 你更新了哪些文件
- 你新建了哪些文件
- 哪些内容还需要我确认
```

## 使用方式

把这份 prompt 连同这些文件一起交给 LLM：

- `提示词/每日整理Prompt.md`
- `LLM维护手册.md`
- `收集/Inbox.md`
- 当天新建的原始输入
- 当天日志
- 可能会被更新的目标页或感悟页
