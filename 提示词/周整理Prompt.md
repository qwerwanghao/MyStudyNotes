# 周整理 Prompt

这份 prompt 只负责“单次执行”。

长期规则、目录约束、frontmatter 规范、允许与禁止事项，统一以 [LLM维护手册.md](../LLM维护手册.md) 为准。

```text
你现在要执行一次“周整理”。

在开始前，先读取并遵循：
- LLM维护手册.md

本次输入范围：
- 最近 7 天的日志：日志/YYYY/YYYY-MM-DD.md
- 最近 7 天新增或更新过的感悟、知识、展望笔记
- 如果已有相关周回顾，也可以一起读取

本次任务目标：
1. 汇总最近 7 天最重要的输入和变化；
2. 生成一份克制、可复核的周回顾；
3. 标记值得继续沉淀的主题；
4. 不删除原始记录，不编造成果。

请按下面步骤执行：
1. 概括本周最重要的 3 到 5 个主题；
2. 归纳：
   - 本周学到了什么
   - 本周最重要的进展
   - 本周卡住的地方
   - 值得升级成长期笔记的主题
   - 下周最重要的三件事
3. 生成或更新本周回顾到 `总结/`
4. 如果本周反复出现某个问题或判断，更新相关的 `感悟/` 或 `展望/`
5. 列出仍然需要我确认的内容

最后输出：
- 你更新了哪些文件
- 你新建了哪些文件
- 哪些内容还需要我确认
```

## 使用方式

把这份 prompt 连同这些文件一起交给 LLM：

- `提示词/周整理Prompt.md`
- `LLM维护手册.md`
- 最近 7 天的日志
- 相关的感悟 / 知识 / 展望
- 可能会被更新的周回顾
