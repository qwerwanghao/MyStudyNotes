# 收集区

> 仓库里摩擦最低的入口。任何还没想清楚、还不值得分类、但怕丢掉的东西，都先放这里。

## 当前入口

- [Inbox.md](Inbox.md)：默认追加入口

## 什么时候写到这里

适合放：
- 临时想法
- 学习摘录
- 问题清单
- 还没想清楚的判断
- 稍后再整理的素材

不必在这里完成：
- 分类
- 命名完美
- 结论提炼
- 长期归档

## 从这里流向哪里

- 更像当天过程和脉络 → [日志/日志.md](../日志/日志.md)
- 更像长期知识 → [知识/知识.md](../知识/知识.md)
- 更像个人判断 → [感悟/感悟.md](../感悟/感悟.md)
- 更像阶段回顾 → [总结/总结.md](../总结/总结.md)
- 更像下一步计划 → [展望/展望.md](../展望/展望.md)
