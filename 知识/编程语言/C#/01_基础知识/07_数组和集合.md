# 数组和集合

## 数组

### 一维数组
1. **数组声明和初始化**
   ```csharp
   // 声明固定大小的数组
   int[] numbers = new int[5];
   
   // 声明并初始化
   int[] scores = new int[] { 85, 92, 78, 95, 88 };
   string[] names = { "Alice", "Bob", "Charlie" };
   ```

2. **数组访问和修改**
   ```csharp
   int[] array = new int[3];
   array[0] = 10;      // 设置元素
   int value = array[0];// 获取元素
   
   // 使用循环访问
   for (int i = 0; i < array.Length; i++)
   {
       array[i] = i * 2;
   }
   ```

### 多维数组
1. **二维数组**
   ```csharp
   // 声明2x3数组
   int[,] matrix = new int[2, 3];
   
   // 初始化二维数组
   int[,] grid = new int[,] {
       { 1, 2, 3 },
       { 4, 5, 6 }
   };
   
   // 访问元素
   matrix[0, 0] = 1;
   int value = grid[1, 2];
   ```

2. **交错数组**
   ```csharp
   // 声明交错数组
   int[][] jaggedArray = new int[3][];
   jaggedArray[0] = new int[] { 1, 2, 3 };
   jaggedArray[1] = new int[] { 4, 5 };
   jaggedArray[2] = new int[] { 6, 7, 8, 9 };
   
   // 访问元素
   int value = jaggedArray[1][0];
   ```

### 数组方法
1. **常用方法**
   ```csharp
   int[] numbers = { 3, 1, 4, 1, 5, 9 };
   
   Array.Sort(numbers);        // 排序
   Array.Reverse(numbers);     // 反转
   int index = Array.IndexOf(numbers, 4);  // 查找元素
   Array.Clear(numbers, 0, 2); // 清除元素
   ```

2. **数组复制**
   ```csharp
   int[] source = { 1, 2, 3, 4, 5 };
   int[] dest = new int[5];
   
   Array.Copy(source, dest, 5);
   // 或者
   source.CopyTo(dest, 0);
   ```

## 集合

### List<T>
1. **创建和初始化**
   ```csharp
   List<int> numbers = new List<int>();
   List<string> names = new List<string>() { "Alice", "Bob" };
   
   // 添加元素
   numbers.Add(10);
   numbers.AddRange(new int[] { 20, 30, 40 });
   ```

2. **常用操作**
   ```csharp
   List<string> items = new List<string>();
   
   items.Add("First");         // 添加
   items.Insert(0, "Start");   // 插入
   items.Remove("First");      // 移除特定元素
   items.RemoveAt(0);          // 移除指定位置的元素
   bool exists = items.Contains("Start"); // 检查是否存在
   ```

### Dictionary<TKey, TValue>
1. **创建和使用**
   ```csharp
   Dictionary<string, int> scores = new Dictionary<string, int>();
   
   // 添加键值对
   scores.Add("Alice", 95);
   scores["Bob"] = 87;
   
   // 访问和修改
   if (scores.ContainsKey("Alice"))
   {
       int score = scores["Alice"];
       scores["Alice"] = 98;
   }
   ```

2. **遍历字典**
   ```csharp
   foreach (KeyValuePair<string, int> pair in scores)
   {
       Console.WriteLine($"{pair.Key}: {pair.Value}");
   }
   
   // 单独遍历键或值
   foreach (string name in scores.Keys) { }
   foreach (int score in scores.Values) { }
   ```

### HashSet<T>
```csharp
HashSet<int> uniqueNumbers = new HashSet<int>();

uniqueNumbers.Add(1);  // 添加元素
uniqueNumbers.Add(2);
uniqueNumbers.Add(1);  // 重复元素不会被添加

bool contains = uniqueNumbers.Contains(1);
uniqueNumbers.Remove(2);
```

### Queue<T>和Stack<T>
1. **队列（Queue）**
   ```csharp
   Queue<string> commandQueue = new Queue<string>();
   
   commandQueue.Enqueue("Move");    // 入队
   commandQueue.Enqueue("Jump");
   string command = commandQueue.Dequeue();  // 出队
   string peek = commandQueue.Peek();// 查看队首元素
   ```

2. **栈（Stack）**
   ```csharp
   Stack<int> numberStack = new Stack<int>();
   
   numberStack.Push(1);     // 压栈
   numberStack.Push(2);
   int value = numberStack.Pop();  // 出栈
   int top = numberStack.Peek();   // 查看栈顶元素
   ```

## Unity中的集合应用

### 游戏对象集合
```csharp
// 管理游戏对象
List<GameObject> activeEnemies = new List<GameObject>();
Dictionary<string, GameObject> prefabCache = new Dictionary<string, GameObject>();

// 对象池
Queue<GameObject> objectPool = new Queue<GameObject>();
```

### 组件集合
```csharp
// 获取所有特定类型的组件
List<Renderer> renderers = new List<Renderer>();
GetComponentsInChildren<Renderer>(renderers);

// 缓存组件引用
Dictionary<string, Transform> waypoints = new Dictionary<string, Transform>();
```

### 集合性能考虑
1. **预分配容量**
   ```csharp
   // 预分配空间避免重新分配
   List<Vector3> positions = new List<Vector3>(100);
   Dictionary<int, Enemy> enemies = new Dictionary<int, Enemy>(50);
   ```

2. **选择合适的集合类型**
   ```csharp
   // 频繁查找使用Dictionary
   Dictionary<int, GameObject> entityMap = new Dictionary<int, GameObject>();
   
   // 唯一性检查使用HashSet
   HashSet<int> visitedNodes = new HashSet<int>();
   
   // 顺序访问使用List
   List<Transform> pathNodes = new List<Transform>();
