# 字符串处理

## 字符串基础

### 字符串创建
1. **基本创建方式**
   ```csharp
   string str1 = "Hello";
   string str2 = new string('*', 5);      // *****
   string str3 = string.Empty;            // 空字符串
   ```

2. **字符串连接**
   ```csharp
   string firstName = "John";
   string lastName = "Doe";
   
   // 使用+运算符
   string fullName1 = firstName + " " + lastName;
   
   // 使用string.Format
   string fullName2 = string.Format("{0} {1}", firstName, lastName);
   
   // 使用字符串插值
   string fullName3 = $"{firstName} {lastName}";
   ```

### 字符串比较
```csharp
string str1 = "hello";
string str2 = "HELLO";

// 区分大小写比较
bool isEqual1 = str1 == str2;                     // false
bool isEqual2 = str1.Equals(str2);                // false

// 忽略大小写比较
bool isEqual3 = str1.Equals(str2, 
    StringComparison.OrdinalIgnoreCase);          // true

// 比较排序顺序
int result = string.Compare(str1, str2);          // > 0
```

## 字符串操作

### 查找和替换
1. **查找操作**
   ```csharp
   string text = "Hello, World!";
   
   // 查找子串
   bool contains = text.Contains("World");         // true
   int index = text.IndexOf("o");                 // 4
   int lastIndex = text.LastIndexOf("o");         // 7
   
   // 开始和结束检查
   bool startsWith = text.StartsWith("Hello");    // true
   bool endsWith = text.EndsWith("!");            // true
   ```

2. **替换操作**
   ```csharp
   string text = "Hello, World!";
   
   // 替换字符或子串
   string newText1 = text.Replace("World", "C#"); // "Hello, C#!"
   string newText2 = text.Replace('o', 'O');      // "HellO, WOrld!"
   ```

### 分割和连接
1. **字符串分割**
   ```csharp
   string data = "apple,banana,orange";
   string[] fruits = data.Split(',');
   
   string text = "Hello World";
   string[] words = text.Split(' ');
   
   // 使用多个分隔符
   string data2 = "apple;banana,orange";
   string[] fruits2 = data2.Split(new[] { ',', ';' });
   ```

2. **字符串连接**
   ```csharp
   string[] words = { "Hello", "World", "!" };
   
   // 使用Join
   string result1 = string.Join(" ", words);
   
   // 使用StringBuilder
   StringBuilder sb = new StringBuilder();
   foreach (string word in words)
   {
       sb.Append(word).Append(" ");
   }
   string result2 = sb.ToString().TrimEnd();
   ```

### 修剪和填充
```csharp
string text = "  Hello, World!  ";

// 修剪空白
string trimmed1 = text.Trim();        // "Hello, World!"
string trimmed2 = text.TrimStart();   // "Hello, World!  "
string trimmed3 = text.TrimEnd();     // "  Hello, World!"

// 填充字符串
string str = "Hello";
string padded1 = str.PadLeft(10);     // "     Hello"
string padded2 = str.PadRight(10, '*');// "Hello*****"
```

## 字符串格式化

### 基本格式化
```csharp
int number = 42;
double price = 12.345;
DateTime date = DateTime.Now;

// 数字格式化
string str1 = number.ToString("D5");          // "00042"
string str2 = price.ToString("F2");           // "12.35"
string str3 = price.ToString("C");            // "¥12.35"

// 日期格式化
string str4 = date.ToString("yyyy-MM-dd");    // "2025-03-02"
string str5 = date.ToString("HH:mm:ss");      // "10:00:00"
```

### 复合格式化
```csharp
// 使用string.Format
string message1 = string.Format("得分：{0:D3}, 时间：{1:F2}秒", 
    42, 12.345);

// 使用字符串插值
int score = 42;
double time = 12.345;
string message2 = $"得分：{score:D3}, 时间：{time:F2}秒";
```

## 字符串性能优化

### StringBuilder使用
```csharp
// 大量字符串连接时使用StringBuilder
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 100; i++)
{
    sb.Append($"Item {i}");
    sb.AppendLine();
}
string result = sb.ToString();
```

### 字符串缓存
```csharp
// 缓存常用字符串
private static readonly string[] CachedMessages = {
    "Game Over",
    "Level Complete",
    "New High Score!"
};

// 使用字符串内存池
string message = string.Intern("Game Over");
```

### 字符串比较优化
```csharp
// 使用StringComparison枚举提高性能
bool equals = string.Equals(
    str1, 
    str2, 
    StringComparison.Ordinal
);

// 避免不必要的大小写转换
bool containsIgnoreCase = str1.IndexOf(
    str2, 
    StringComparison.OrdinalIgnoreCase
) >= 0;
