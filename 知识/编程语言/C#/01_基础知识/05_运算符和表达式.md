# 运算符和表达式

## 算术运算符

### 基本算术运算符
1. **基本运算**
   ```csharp
   int a = 10, b = 3;
   
   int sum = a + b;      // 加法：13
   int diff = a - b;     // 减法：7
   int product = a * b;  // 乘法：30
   int quotient = a / b; // 整除：3
   int remainder = a % b;// 取余：1
   ```

2. **自增自减**
   ```csharp
   int x = 5;
   x++;                  // 后自增：x = 6
   ++x;                  // 前自增：x = 7
   x--;                  // 后自减：x = 6
   --x;                  // 前自减：x = 5
   ```

### 复合赋值运算符
```csharp
int n = 10;
n += 5;                  // n = n + 5
n -= 3;                  // n = n - 3
n *= 2;                  // n = n * 2
n /= 4;                  // n = n / 4
n %= 3;                  // n = n % 3
```

## 比较运算符

### 基本比较
```csharp
int x = 5, y = 10;

bool isEqual = x == y;           // 等于
bool isNotEqual = x != y;        // 不等于
bool isGreater = x > y;          // 大于
bool isLess = x < y;             // 小于
bool isGreaterOrEqual = x >= y;  // 大于等于
bool isLessOrEqual = x <= y;     // 小于等于
```

### 对象比较
```csharp
string str1 = "hello";
string str2 = "hello";

bool areEqual = str1.Equals(str2);    // 对象内容比较
bool areSame = ReferenceEquals(str1, str2);  // 引用比较
```

## 逻辑运算符

### 基本逻辑运算符
```csharp
bool a = true, b = false;

bool andResult = a && b;    // 逻辑与
bool orResult = a || b;     // 逻辑或
bool notResult = !a;        // 逻辑非
```

### 短路逻辑
```csharp
// 短路与
if (CheckCondition() && ProcessData())
{
    // 只有当CheckCondition()为true时才会执行ProcessData()
}

// 短路或
if (ValidateCache() || LoadFromDatabase())
{
    // 只有当ValidateCache()为false时才会执行LoadFromDatabase()
}
```

## 位运算符

### 基本位运算
```csharp
int x = 5;     // 二进制：0101
int y = 3;     // 二进制：0011

int and = x & y;   // 位与：0001
int or = x | y;    // 位或：0111
int xor = x ^ y;   // 位异或：0110
int not = ~x;      // 位取反：1010 (补码形式)
```

### 移位运算
```csharp
int num = 8;       // 二进制：1000
int leftShift = num << 2;   // 左移2位：100000 (32)
int rightShift = num >> 1;  // 右移1位：0100 (4)
```

## 条件运算符

### 三元运算符
```csharp
int age = 20;
string status = age >= 18 ? "成年" : "未成年";

// 嵌套使用
int score = 75;
string grade = score >= 90 ? "A" : 
               score >= 80 ? "B" : 
               score >= 70 ? "C" : "D";
```

### null条件运算符
```csharp
string name = null;
int? length = name?.Length;  // 如果name为null，则length为null

// 链式使用
var value = obj?.GetComponent<Renderer>()?.material?.color;
```

## Unity中的常用运算

### Vector运算
```csharp
Vector3 pos1 = new Vector3(1, 0, 0);
Vector3 pos2 = new Vector3(0, 1, 0);

// 向量加法
Vector3 sum = pos1 + pos2;

// 向量乘法
Vector3 scaled = pos1 * 2f;

// 点乘
float dot = Vector3.Dot(pos1, pos2);

// 叉乘
Vector3 cross = Vector3.Cross(pos1, pos2);
```

### Quaternion运算
```csharp
Quaternion rot1 = Quaternion.Euler(0, 90, 0);
Quaternion rot2 = Quaternion.Euler(0, 45, 0);

// 四元数乘法（旋转组合）
Quaternion combined = rot1 * rot2;

// 旋转插值
Quaternion interpolated = Quaternion.Lerp(rot1, rot2, 0.5f);
```

## 运算符优先级

### 优先级顺序
1. **初级运算符**
   - ()、[]、.、?.、?[]
   
2. **一元运算符**
   - +、-、!、~、++、--、(type)

3. **乘除运算符**
   - *、/、%

4. **加减运算符**
   - +、-

5. **移位运算符**
   - <<、>>

6. **关系运算符**
   - <、>、<=、>=、is、as

7. **相等运算符**
   - ==、!=

8. **位运算符**
   - &、^、|

9. **逻辑运算符**
   - &&、||

10. **条件运算符**
    - ?:

11. **赋值运算符**
    - =、+=、-=、*=、/=、%=、&=、|=、^=、<<=、>>=

### 使用括号明确优先级
```csharp
int result = (a + b) * c;    // 确保加法先执行
bool logic = (a > b) && (c < d);  // 明确逻辑组合
Vector3 position = transform.position + (Vector3.up * jumpForce);
