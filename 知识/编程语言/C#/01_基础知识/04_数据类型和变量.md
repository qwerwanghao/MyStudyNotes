# 数据类型和变量

## 值类型

### 整数类型
1. **基本整数类型**
   ```csharp
   sbyte  // 8位有符号整数  (-128 到 127)
   byte   // 8位无符号整数  (0 到 255)
   short  // 16位有符号整数 (-32,768 到 32,767)
   ushort // 16位无符号整数 (0 到 65,535)
   int    // 32位有符号整数 (-2^31 到 2^31-1)
   uint   // 32位无符号整数 (0 到 2^32-1)
   long   // 64位有符号整数 (-2^63 到 2^63-1)
   ulong  // 64位无符号整数 (0 到 2^64-1)
   ```

2. **使用示例**
   ```csharp
   int age = 25;
   long population = 7800000000L;
   byte level = 100;
   ```

### 浮点类型
1. **浮点数类型**
   ```csharp
   float  // 32位单精度浮点数 (±1.5 × 10^-45 到 ±3.4 × 10^38)
   double // 64位双精度浮点数 (±5.0 × 10^-324 到 ±1.7 × 10^308)
   decimal // 128位高精度十进制数
   ```

2. **使用示例**
   ```csharp
   float speed = 12.5f;
   double distance = 123.456;
   decimal price = 19.99m;
   ```

### 布尔类型
```csharp
bool isActive = true;
bool isCompleted = false;
```

### 字符类型
```csharp
char grade = 'A';
char symbol = '★';
```

## 引用类型

### 字符串类型
1. **声明和初始化**
   ```csharp
   string name = "Alice";
   string message = string.Empty; // 空字符串
   string nullString = null;     // null字符串
   ```

2. **字符串操作**
   ```csharp
   string firstName = "John";
   string lastName = "Doe";
   string fullName = firstName + " " + lastName;
   string interpolated = $"{firstName} {lastName}";
   ```

### 对象类型
1. **Object类**
   ```csharp
   object obj1 = 100;        // 装箱
   object obj2 = "Hello";    // 引用类型装箱
   int num = (int)obj1;      // 拆箱
   ```

2. **动态类型**
   ```csharp
   dynamic dynVar = 100;
   dynVar = "Hello";    // 可以改变类型
   ```

## 变量声明与初始化

### 变量声明
1. **基本声明**
   ```csharp
   int count;           // 声明变量
   string name;
   bool isValid;
   ```

2. **声明并初始化**
   ```csharp
   int age = 25;
   string message = "Hello";
   double price = 99.99;
   ```

### 变量作用域
1. **局部变量**
   ```csharp
   void MyMethod()
   {
       int x = 10;     // 局部变量
       {
           int y = 20; // 块级作用域
       }
       // y在这里不可访问
   }
   ```

2. **成员变量**
   ```csharp
   public class Player
   {
       private int health = 100;   // 私有成员变量
       public string playerName;   // 公有成员变量
       protected float speed;      // 保护成员变量
   }
   ```

## 类型转换

### 隐式转换
```csharp
int num = 100;
long bigNum = num;      // int到long的隐式转换
float f = 100;         // int到float的隐式转换
```

### 显式转换
```csharp
double d = 123.45;
int i = (int)d;        // 显式转换，小数部分被截断

string numStr = "123";
int parsed = int.Parse(numStr);    // 字符串转整数
double parsed2 = double.Parse("123.45"); // 字符串转double
```

### 转换方法
```csharp
int number = 789;
string str = number.ToString();    // 转换为字符串
string hex = number.ToString("X"); // 转换为十六进制字符串

// Convert类方法
int intValue = Convert.ToInt32("123");
bool boolValue = Convert.ToBoolean(1);
```
