# 基本语法

## 程序结构

### 基本组成
1. **命名空间（Namespace）**
   ```csharp
   namespace MyProgram
   {
       // 命名空间内容
   }
   ```

2. **类（Class）**
   ```csharp
   public class Program
   {
       // 类的成员
   }
   ```

3. **主方法（Main）**
   ```csharp
   static void Main(string[] args)
   {
       // 程序入口点
   }
   ```

### 代码规范
1. **命名规则**
   - 类名：首字母大写（PascalCase）
   - 方法名：首字母大写（PascalCase）
   - 变量名：首字母小写（camelCase）
   - 常量名：全大写（UPPER_CASE）

2. **缩进和空格**
   - 使用4个空格缩进
   - 大括号单独成行
   - 运算符两边加空格
   - 逗号后加空格

## 注释写法

### 单行注释
```csharp
// 这是单行注释
int x = 5; // 行尾注释
```

### 多行注释
```csharp
/* 这是多行注释
   可以跨越多行
   便于大段说明 */
```

### 文档注释
```csharp
/// <summary>
/// 方法的功能说明
/// </summary>
/// <param name="x">参数x的说明</param>
/// <returns>返回值的说明</returns>
public int MyMethod(int x)
{
    return x * 2;
}
```

## 基本语句

### 声明语句
1. **变量声明**
   ```csharp
   int number = 10;
   string name = "John";
   bool isValid = true;
   ```

2. **常量声明**
   ```csharp
   const double PI = 3.14159;
   const string APP_NAME = "MyApp";
   ```

### 表达式语句
1. **赋值表达式**
   ```csharp
   x = 100;
   y = x + 50;
   ```

2. **方法调用**
   ```csharp
   Console.WriteLine("Hello");
   Math.Pow(2, 3);
   ```

### 控制语句
1. **条件语句**
   ```csharp
   if (score >= 60)
   {
       Console.WriteLine("通过");
   }
   else
   {
       Console.WriteLine("未通过");
   }
   ```

2. **循环语句**
   ```csharp
   for (int i = 0; i < 10; i++)
   {
       // 循环体
   }

   while (condition)
   {
       // 循环体
   }
   ```

## 基本输入输出

### 控制台输出
1. **基本输出**
   ```csharp
   Console.Write("不换行输出");
   Console.WriteLine("换行输出");
   ```

2. **格式化输出**
   ```csharp
   string name = "Alice";
   int age = 20;
   Console.WriteLine($"姓名：{name}，年龄：{age}");
   Console.WriteLine("姓名：{0}，年龄：{1}", name, age);
   ```

### 控制台输入
1. **读取字符串**
   ```csharp
   string input = Console.ReadLine();
   ```

2. **读取数字**
   ```csharp
   int number = int.Parse(Console.ReadLine());
   double value = Convert.ToDouble(Console.ReadLine());
   ```

## Unity中的特殊语法

### MonoBehaviour方法
1. **生命周期方法**
   ```csharp
   void Start()
   {
       // 初始化代码
   }

   void Update()
   {
       // 每帧执行的代码
   }
   ```

2. **事件方法**
   ```csharp
   void OnCollisionEnter(Collision collision)
   {
       // 碰撞处理代码
   }

   void OnTriggerEnter(Collider other)
   {
       // 触发器处理代码
   }
   ```

### Unity特有语法
1. **协程**
   ```csharp
   IEnumerator MyCoroutine()
   {
       yield return new WaitForSeconds(1f);
       // 延迟执行的代码
   }
   ```

2. **序列化字段**
   ```csharp
   [SerializeField]
   private float speed = 5f;

   public GameObject prefab;
   ```

## 代码组织

### 访问修饰符
1. **常用修饰符**
   - public：公开访问
   - private：私有访问
   - protected：保护访问
   - internal：程序集访问

2. **使用示例**
   ```csharp
   public class Player
   {
       private int health;
       protected float speed;
       public string name;
   }
   ```

### 代码文件组织
1. **文件结构**
   ```csharp
   // 引用命名空间
   using UnityEngine;
   using System.Collections;

   // 命名空间声明
   namespace GameProject
   {
       // 类定义
       public class GameManager
       {
           // 类成员
       }
   }
   ```

2. **最佳实践**
   - 每个类放在单独的文件中
   - 文件名与类名相同
   - 相关类放在同一命名空间
   - 使用适当的文件夹组织
