## 1. 粒子系统概述

粒子系统是Unity中用于模拟流体、烟雾、火焰、爆炸等复杂视觉效果的强大工具。它通过生成和控制大量小型图像（粒子）来创建这些效果。

### 1.1 粒子系统的基本原理

- **粒子**：具有生命周期的小型图像或网格
- **发射器**：控制粒子的生成位置和方式
- **模块化系统**：通过不同模块控制粒子的各种属性
- **GPU加速**：利用图形处理器加速大量粒子的渲染

### 1.2 粒子系统的应用场景

- 环境效果：雨、雪、雾、云
- 火焰和烟雾效果
- 魔法和技能特效
- 爆炸和冲击波
- 拖尾效果
- 用户界面反馈

## 2. 创建和配置粒子系统

### 2.1 创建粒子系统

```
GameObject → Effects → Particle System
```

或通过代码创建：

```csharp
GameObject particleObj = new GameObject("MyParticleSystem");
ParticleSystem particleSystem = particleObj.AddComponent<ParticleSystem>();
```

### 2.2 粒子系统组件

- **Particle System**：主组件，控制粒子的生成和基本属性
- **Particle System Renderer**：控制粒子的渲染方式

### 2.3 粒子系统预览窗口

- 实时预览效果
- 模拟和暂停功能
- 时间控制
- 重启系统

## 3. 粒子系统主模块

主模块控制粒子系统的基本行为和属性。

### 3.1 持续时间和循环

- **Duration**：粒子系统的持续时间
- **Looping**：是否循环播放
- **Prewarm**：预热系统（第一次播放时已经模拟了一个周期）
- **Start Delay**：开始延迟时间

### 3.2 粒子生成设置

- **Start Lifetime**：粒子的生命周期
- **Start Speed**：粒子的初始速度
- **Start Size**：粒子的初始大小
- **Start Rotation**：粒子的初始旋转
- **Start Color**：粒子的初始颜色
- **Gravity Modifier**：重力影响
- **Simulation Space**：模拟空间（局部/世界/自定义）
- **Max Particles**：最大粒子数量

### 3.3 随机化设置

每个属性都可以设置为：
- 常量值
- 两个值之间的随机值
- 曲线值
- 两条曲线之间的随机值

## 4. 粒子系统关键模块

### 4.1 Emission（发射）模块

控制粒子的发射速率和时机。

- **Rate over Time**：每秒发射的粒子数
- **Rate over Distance**：每单位距离发射的粒子数
- **Bursts**：爆发式发射
  - Time：爆发时间点
  - Count：爆发粒子数量
  - Cycles：爆发循环次数
  - Interval：爆发间隔

### 4.2 Shape（形状）模块

定义粒子的发射区域和方向。

- **Shape Types**：
  - Sphere/Hemisphere：球体/半球体
  - Cone：锥体
  - Box：盒子
  - Circle/Edge：圆形/边缘
  - Mesh：网格
  - Mesh Renderer/Skinned Mesh Renderer：渲染器网格

- **Emission Settings**：
  - Radius：半径
  - Angle：角度
  - Randomize Direction：随机化方向
  - Spherize Direction：球面化方向

### 4.3 Velocity over Lifetime（生命周期速度）模块

控制粒子在生命周期内的速度变化。

- **Linear**：线性速度
- **Orbital**：轨道速度
- **Radial**：径向速度
- **Speed Modifier**：速度修改器

### 4.4 Color over Lifetime（生命周期颜色）模块

控制粒子在生命周期内的颜色变化。

- **Color**：颜色渐变
- **Alpha**：透明度变化

### 4.5 Size over Lifetime（生命周期大小）模块

控制粒子在生命周期内的大小变化。

- **Size**：大小曲线
- **Separate Axes**：分轴控制（X、Y、Z）

### 4.6 Rotation over Lifetime（生命周期旋转）模块

控制粒子在生命周期内的旋转变化。

- **Angular Velocity**：角速度
- **Separate Axes**：分轴旋转

### 4.7 Collision（碰撞）模块

使粒子能够与场景中的物体发生碰撞。

- **Type**：
  - Planes：平面碰撞
  - World：世界碰撞
  - Custom Planes：自定义平面

- **Collision Settings**：
  - Dampen：阻尼
  - Bounce：反弹
  - Lifetime Loss：生命损失

### 4.8 Sub Emitters（子发射器）模块

允许粒子在特定事件（如出生、死亡或碰撞）时生成其他粒子系统。

- **Birth**：粒子出生时触发
- **Death**：粒子死亡时触发
- **Collision**：粒子碰撞时触发

### 4.9 Texture Sheet Animation（纹理表动画）模块

使用精灵表为粒子添加动画。

- **Tiles**：X和Y方向的瓦片数
- **Animation**：
  - Whole Sheet：整个表格
  - Single Row：单行
  - Random Row：随机行
- **Frame over Time**：随时间变化的帧

## 5. 粒子系统渲染器

### 5.1 渲染模式

- **Billboard**：始终面向摄像机
- **Stretched**：拉伸粒子
- **Horizontal/Vertical Billboard**：水平/垂直公告板
- **Mesh**：使用3D网格作为粒子

### 5.2 材质和着色器

- **Material**：粒子使用的材质
- **Particle Shaders**：
  - Additive：加法混合，适合发光效果
  - Alpha Blended：透明混合，适合烟雾
  - Multiply：乘法混合，适合阴影
  - Unlit：不受光照影响

### 5.3 渲染优化

- **Sorting**：排序方式
- **Render Queue**：渲染队列
- **Cast/Receive Shadows**：投射/接收阴影
- **Max Particle Size**：最大粒子尺寸

## 6. 粒子系统脚本控制

### 6.1 基本控制

```csharp
// 获取粒子系统引用
ParticleSystem ps = GetComponent<ParticleSystem>();

// 播放
ps.Play();

// 停止
ps.Stop();

// 暂停
ps.Pause();

// 清除所有粒子
ps.Clear();

// 检查是否正在播放
bool isPlaying = ps.isPlaying;

// 获取当前粒子数量
int particleCount = ps.particleCount;
```

### 6.2 修改粒子系统属性

```csharp
// 获取主模块
var main = ps.main;

// 修改持续时间
main.duration = 2.0f;

// 修改循环
main.loop = true;

// 修改起始速度
main.startSpeed = 5.0f;

// 修改起始颜色
main.startColor = Color.red;

// 获取发射模块
var emission = ps.emission;

// 修改发射速率
emission.rateOverTime = 10.0f;

// 获取形状模块
var shape = ps.shape;

// 修改形状类型
shape.shapeType = ParticleSystemShapeType.Sphere;

// 修改半径
shape.radius = 1.5f;
```

### 6.3 粒子操作

```csharp
// 创建粒子数组
ParticleSystem.Particle[] particles = new ParticleSystem.Particle[ps.particleCount];

// 获取所有粒子
int count = ps.GetParticles(particles);

// 修改粒子属性
for (int i = 0; i < count; i++)
{
    // 修改位置
    particles[i].position += Vector3.up * Time.deltaTime;
    
    // 修改大小
    particles[i].startSize *= 1.01f;
    
    // 修改颜色
    particles[i].startColor = Color.Lerp(Color.yellow, Color.red, particles[i].remainingLifetime / particles[i].startLifetime);
}

// 应用修改
ps.SetParticles(particles, count);
```

## 7. 高级粒子效果

### 7.1 拖尾效果

使用Trail模块或TrailRenderer组件创建粒子拖尾。

- **Trail Module**：
  - Ratio：产生拖尾的粒子比例
  - Lifetime：拖尾生命周期
  - Min/Max Vertex Distance：顶点之间的最小/最大距离
  - Width over Trail：拖尾宽度曲线
  - Color over Trail**：拖尾颜色渐变

### 7.2 光照效果

- **Lights Module**：
  - Ratio：产生光源的粒子比例
  - Light Range：光照范围
  - Intensity：光照强度
  - Color：光照颜色

### 7.3 噪声效果

使用Noise模块添加随机运动。

- **Strength**：噪声强度
- **Frequency**：噪声频率
- **Scroll Speed**：滚动速度
- **Damping**：阻尼

### 7.4 VFX Graph（Unity 2019+）

Visual Effect Graph是Unity的高级粒子系统，使用节点图形界面创建复杂效果。

- 基于GPU的高性能系统
- 支持数百万粒子
- 节点式编程界面
- 与HDRP渲染管线集成

## 8. 粒子系统优化

### 8.1 性能考虑

- **粒子数量**：减少最大粒子数
- **更新频率**：降低不必要的更新频率
- **LOD系统**：基于距离使用不同复杂度的粒子系统
- **预热**：使用Prewarm避免突然出现
- **共享材质**：使用相同的材质减少批次

### 8.2 移动平台优化

- 减少过度重叠的透明粒子
- 使用更简单的着色器
- 限制碰撞检测
- 减小纹理尺寸
- 禁用不必要的模块

### 8.3 批处理优化

```csharp
// 禁用自动批处理以手动控制
ParticleSystemRenderer renderer = GetComponent<ParticleSystemRenderer>();
renderer.enableGPUInstancing = true;
```

## 9. 实战案例

### 9.1 火焰效果

```csharp
public class FireEffect : MonoBehaviour
{
    private ParticleSystem firePS;
    private ParticleSystem smokePS;
    
    [Range(0.1f, 5f)]
    public float fireIntensity = 1f;
    
    void Start()
    {
        // 获取火焰粒子系统
        firePS = transform.Find("Fire").GetComponent<ParticleSystem>();
        smokePS = transform.Find("Smoke").GetComponent<ParticleSystem>();
        
        // 初始化
        UpdateFireIntensity();
    }
    
    public void UpdateFireIntensity()
    {
        // 修改火焰发射率
        var fireEmission = firePS.emission;
        fireEmission.rateOverTime = 20f * fireIntensity;
        
        // 修改火焰大小
        var fireMain = firePS.main;
        fireMain.startSize = new ParticleSystem.MinMaxCurve(0.5f * fireIntensity, 1.5f * fireIntensity);
        
        // 修改烟雾发射率
        var smokeEmission = smokePS.emission;
        smokeEmission.rateOverTime = 5f * fireIntensity;
    }
    
    // 响应外部事件（如风或燃料变化）
    public void OnWindChange(float windStrength, Vector3 windDirection)
    {
        // 修改火焰和烟雾的速度
        var fireVelocity = firePS.velocityOverLifetime;
        fireVelocity.x = windStrength * windDirection.x;
        fireVelocity.z = windStrength * windDirection.z;
        
        var smokeVelocity = smokePS.velocityOverLifetime;
        smokeVelocity.x = windStrength * 2f * windDirection.x;
        smokeVelocity.z = windStrength * 2f * windDirection.z;
    }
}
```

### 9.2 魔法技能效果

```csharp
public class MagicSpell : MonoBehaviour
{
    public ParticleSystem chargeEffect;
    public ParticleSystem castEffect;
    public ParticleSystem impactEffect;
    
    public float chargeDuration = 2f;
    public float spellSpeed = 10f;
    
    private bool isCharging = false;
    private float chargeTime = 0f;
    
    void Update()
    {
        // 按下鼠标左键开始充能
        if (Input.GetMouseButtonDown(0) && !isCharging)
        {
            StartCharging();
        }
        
        // 充能中
        if (isCharging)
        {
            chargeTime += Time.deltaTime;
            
            // 更新充能效果
            var chargeMain = chargeEffect.main;
            chargeMain.startColor = Color.Lerp(Color.blue, Color.red, chargeTime / chargeDuration);
            
            // 释放鼠标左键释放技能
            if (Input.GetMouseButtonUp(0))
            {
                CastSpell();
            }
        }
    }
    
    void StartCharging()
    {
        isCharging = true;
        chargeTime = 0f;
        chargeEffect.Play();
    }
    
    void CastSpell()
    {
        isCharging = false;
        chargeEffect.Stop();
        
        // 计算技能威力（基于充能时间）
        float powerRatio = Mathf.Clamp01(chargeTime / chargeDuration);
        
        // 创建并发射技能
        GameObject spellObj = new GameObject("MagicSpell");
        spellObj.transform.position = transform.position + transform.forward;
        
        // 添加粒子系统
        ParticleSystem spellPS = Instantiate(castEffect, spellObj.transform);
        
        // 调整粒子系统参数基于充能
        var main = spellPS.main;
        main.startSize = new ParticleSystem.MinMaxCurve(0.5f + powerRatio, 1f + powerRatio);
        main.startColor = Color.Lerp(Color.blue, Color.red, powerRatio);
        
        // 添加移动和碰撞逻辑
        SpellProjectile projectile = spellObj.AddComponent<SpellProjectile>();
        projectile.speed = spellSpeed;
        projectile.power = powerRatio;
        projectile.impactEffect = impactEffect;
    }
}

public class SpellProjectile : MonoBehaviour
{
    public float speed = 10f;
    public float power = 1f;
    public float lifetime = 5f;
    public ParticleSystem impactEffect;
    
    private float timer = 0f;
    
    void Update()
    {
        // 移动
        transform.Translate(Vector3.forward * speed * Time.deltaTime);
        
        // 生命周期
        timer += Time.deltaTime;
        if (timer >= lifetime)
        {
            Destroy(gameObject);
        }
    }
    
    void OnTriggerEnter(Collider other)
    {
        // 创建碰撞效果
        ParticleSystem impact = Instantiate(impactEffect, transform.position, Quaternion.identity);
        
        // 调整碰撞效果基于威力
        var main = impact.main;
        main.startSize = new ParticleSystem.MinMaxCurve(1f * power, 2f * power);
        
        // 对目标造成伤害
        IDamageable target = other.GetComponent<IDamageable>();
        if (target != null)
        {
            target.TakeDamage(10f * power);
        }
        
        // 销毁投射物
        Destroy(gameObject);
    }
}
```

### 9.3 环境效果系统

```csharp
public class WeatherSystem : MonoBehaviour
{
    public ParticleSystem rainPS;
    public ParticleSystem snowPS;
    public ParticleSystem fogPS;
    
    [Range(0f, 1f)]
    public float rainIntensity = 0f;
    
    [Range(0f, 1f)]
    public float snowIntensity = 0f;
    
    [Range(0f, 1f)]
    public float fogIntensity = 0f;
    
    void Start()
    {
        UpdateWeather();
    }
    
    public void UpdateWeather()
    {
        // 更新雨
        var rainEmission = rainPS.emission;
        rainEmission.rateOverTime = 1000f * rainIntensity;
        
        if (rainIntensity > 0)
            rainPS.Play();
        else
            rainPS.Stop();
            
        // 更新雪
        var snowEmission = snowPS.emission;
        snowEmission.rateOverTime = 500f * snowIntensity;
        
        if (snowIntensity > 0)
            snowPS.Play();
        else
            snowPS.Stop();
            
        // 更新雾
        var fogEmission = fogPS.emission;
        fogEmission.rateOverTime = 20f * fogIntensity;
        
        var fogMain = fogPS.main;
        fogMain.startLifetime = 5f + 15f * fogIntensity;
        
        if (fogIntensity > 0)
            fogPS.Play();
        else
            fogPS.Stop();
    }
    
    // 平滑过渡到新天气
    public IEnumerator TransitionWeather(float targetRain, float targetSnow, float targetFog, float duration)
    {
        float startRain = rainIntensity;
        float startSnow = snowIntensity;
        float startFog = fogIntensity;
        float timer = 0f;
        
        while (timer < duration)
        {
            timer += Time.deltaTime;
            float t = timer / duration;
            
            rainIntensity = Mathf.Lerp(startRain, targetRain, t);
            snowIntensity = Mathf.Lerp(startSnow, targetSnow, t);
            fogIntensity = Mathf.Lerp(startFog, targetFog, t);
            
            UpdateWeather();
            
            yield return null;
        }
        
        // 确保最终值精确
        rainIntensity = targetRain;
        snowIntensity = targetSnow;
        fogIntensity = targetFog;
        UpdateWeather();
    }
}
```
