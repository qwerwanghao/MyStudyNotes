# Unity导航系统

## 1. 导航系统概述

Unity的导航系统（Navigation System）是一个用于实现AI角色自动寻路和导航的强大工具。它主要由NavMesh（导航网格）、NavMeshAgent（导航代理）和NavMeshObstacle（导航障碍物）等组件组成。

### 1.1 主要组件

- **NavMesh**：场景中可行走区域的网格表示
- **NavMeshAgent**：在NavMesh上进行寻路的AI代理
- **NavMeshObstacle**：动态障碍物，可以改变NavMesh
- **NavMeshSurface**：用于烘焙NavMesh的组件
- **Off-Mesh Link**：连接不相连区域的链接点

### 1.2 工作流程

```
场景设置 → 标记可行走区域 → 烘焙NavMesh → 添加NavMeshAgent → 实现导航逻辑
```

## 2. NavMesh系统设置

### 2.1 场景准备

- 标记静态物体为Static
- 设置地形和障碍物的Layer
- 配置Navigation窗口参数

### 2.2 NavMesh烘焙设置

- **Agent Size**：代理尺寸
  - Radius：半径
  - Height：高度
  - Step Height：可跨越高度
  - Max Slope：最大坡度

- **Area Types**：区域类型
  - Walkable：可行走区域
  - Not Walkable：不可行走区域
  - Jump：跳跃区域
  - 自定义区域类型

### 2.3 高级设置


## 3. NavMeshAgent组件

### 3.1 基本属性

- **Steering**：转向设置
  - Speed：移动速度
  - Angular Speed：转向速度
  - Acceleration：加速度
  - Stopping Distance：停止距离

- **Obstacle Avoidance**：避障设置
  - Radius：避障半径
  - Height：避障高度
  - Quality：避障质量
  - Priority：避障优先级

- **Path Finding**：寻路设置
  - Auto Traverse OffMesh Link：自动穿越链接点
  - Auto Brake：自动刹车
  - Auto Repath：自动重新寻路

### 3.2 基本使用

```csharp
public class AIController : MonoBehaviour
{
    private NavMeshAgent agent;
    public Transform target;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
    }
    
    void Update()
    {
        if (target != null)
        {
            // 设置目标位置
            agent.SetDestination(target.position);
            
            // 检查是否到达目标
            if (!agent.pathPending && agent.remainingDistance <= agent.stoppingDistance)
            {
                // 已到达目标
                OnReachDestination();
            }
        }
    }
    
    void OnReachDestination()
    {
        Debug.Log("已到达目标位置");
    }
}
```

### 3.3 路径控制

```csharp
public class PathController : MonoBehaviour
{
    private NavMeshAgent agent;
    private NavMeshPath path;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        path = new NavMeshPath();
    }
    
    // 计算路径但不移动
    public bool CalculatePath(Vector3 destination)
    {
        return NavMesh.CalculatePath(transform.position, destination, NavMesh.AllAreas, path);
    }
    
    // 显示路径
    void OnDrawGizmos()
    {
        if (path != null && path.corners.Length > 0)
        {
            // 绘制路径线
            Gizmos.color = Color.yellow;
            for (int i = 0; i < path.corners.Length - 1; i++)
            {
                Gizmos.DrawLine(path.corners[i], path.corners[i + 1]);
            }
        }
    }
    
    // 获取路径长度
    public float GetPathLength()
    {
        float length = 0;
        for (int i = 0; i < path.corners.Length - 1; i++)
        {
            length += Vector3.Distance(path.corners[i], path.corners[i + 1]);
        }
        return length;
    }
}
```

### 5.2 Off-Mesh Links

```csharp
public class CustomOffMeshLink : MonoBehaviour
{
    private OffMeshLink link;
    
    void Start()
    {
        link = GetComponent<OffMeshLink>();
        
        // 配置链接
        link.startTransform = transform.Find("Start");
        link.endTransform = transform.Find("End");
        link.costOverride = 1.5f; // 设置使用成本
        link.biDirectional = true; // 双向链接
        link.activated = true; // 激活链接
    }
    
    // 自定义链接过渡动画
    IEnumerator CustomLinkTraversal(NavMeshAgent agent)
    {
        // 禁用agent的移动
        agent.enabled = false;
        
        // 获取起点和终点
        Vector3 startPos = link.startTransform.position;
        Vector3 endPos = link.endTransform.position;
        
        // 执行跳跃动画
        float time = 0;
        while (time < 1)
        {
            time += Time.deltaTime * 2; // 2秒完成过渡
            
            // 抛物线运动
            float height = 2f * Mathf.Sin(time * Mathf.PI);
            Vector3 pos = Vector3.Lerp(startPos, endPos, time);
            pos.y += height;
            
            agent.transform.position = pos;
            yield return null;
        }
        
        // 重新启用agent
        agent.enabled = true;
    }
}
```

### 5.3 群体移动

```csharp
public class GroupMovement : MonoBehaviour
{
    public List<NavMeshAgent> agents = new List<NavMeshAgent>();
    public float groupRadius = 5f;
    
    // 群体移动到目标
    public void MoveGroupTo(Vector3 target)
    {
        // 计算每个智能体的目标位置
        for (int i = 0; i < agents.Count; i++)
        {
            float angle = i * (360f / agents.Count);
            Vector3 offset = Quaternion.Euler(0, angle, 0) * Vector3.forward * groupRadius;
            Vector3 agentTarget = target + offset;
            
            agents[i].SetDestination(agentTarget);
        }
    }
    
    // 保持队形
    void UpdateFormation()
    {
        Vector3 centerPos = GetGroupCenter();
        
        foreach (var agent in agents)
        {
            // 计算理想位置
            Vector3 idealPos = agent.transform.position - centerPos;
            idealPos = idealPos.normalized * groupRadius;
            idealPos += centerPos;
            
            // 调整速度以保持队形
            Vector3 moveDirection = (idealPos - agent.transform.position);
            if (moveDirection.magnitude > 0.1f)
            {
                agent.velocity += moveDirection.normalized * Time.deltaTime;
            }
        }
    }
    
    Vector3 GetGroupCenter()
    {
        Vector3 center = Vector3.zero;
        foreach (var agent in agents)
        {
            center += agent.transform.position;
        }
        return center / agents.Count;
    }
}
```

## 6. 导航网格生成

### 6.1 运行时生成NavMesh

```csharp
public class RuntimeNavMeshBuilder : MonoBehaviour
{
    public LayerMask walkableMask;
    public float agentRadius = 0.5f;
    public float agentHeight = 2f;
    
    public void BuildNavMesh()
    {
        // 创建NavMesh构建设置
        var settings = new NavMeshBuildSettings();
        settings.agentRadius = agentRadius;
        settings.agentHeight = agentHeight;
        
        // 收集场景几何体
        var sources = new List<NavMeshBuildSource>();
        var markups = new List<NavMeshBuildMarkup>();
        
        NavMeshBuilder.CollectSources(
            transform, // 根节点
            walkableMask, // 可行走层
            NavMeshCollectGeometry.RenderMeshes, // 收集类型
            0, // 区域类型
            markups, // 标记
            sources // 输出源
        );
        
        // 构建NavMesh
        var bounds = new Bounds(transform.position, Vector3.one * 1000);
        var data = NavMeshBuilder.BuildNavMeshData(
            settings,
            sources,
            bounds,
            transform.position,
            Quaternion.identity
        );
        
        // 应用NavMesh
        NavMesh.AddNavMeshData(data);
    }
}
```

### 6.2 NavMesh分块加载

```csharp
public class NavMeshTileManager : MonoBehaviour
{
    private Dictionary<Vector2Int, NavMeshData> tiles = new Dictionary<Vector2Int, NavMeshData>();
    public float tileSize = 100f;
    
    // 加载区块
    public void LoadTile(Vector2Int tileCoord)
    {
        if (!tiles.ContainsKey(tileCoord))
        {
            // 计算区块边界
            Vector3 tileCenter = new Vector3(
                tileCoord.x * tileSize,
                0,
                tileCoord.y * tileSize
            );
            
            Bounds tileBounds = new Bounds(
                tileCenter,
                new Vector3(tileSize, 1000f, tileSize)
            );
            
            // 构建NavMesh
            var sources = new List<NavMeshBuildSource>();
            NavMeshBuilder.CollectSources(
                tileBounds,
                LayerMask.GetMask("Ground"),
                NavMeshCollectGeometry.RenderMeshes,
                0,
                new List<NavMeshBuildMarkup>(),
                sources
            );
            
            var data = NavMeshBuilder.BuildNavMeshData(
                NavMesh.GetSettingsByID(0),
                sources,
                tileBounds,
                Vector3.zero,
                Quaternion.identity
            );
            
            tiles.Add(tileCoord, data);
            NavMesh.AddNavMeshData(data);
        }
    }
    
    // 卸载区块
    public void UnloadTile(Vector2Int tileCoord)
    {
        if (tiles.TryGetValue(tileCoord, out NavMeshData data))
        {
            NavMesh.RemoveNavMeshData(data);
            tiles.Remove(tileCoord);
        }
    }
    
    // 根据玩家位置更新区块
    public void UpdateTilesAroundPlayer(Vector3 playerPosition)
    {
        Vector2Int playerTile = new Vector2Int(
            Mathf.FloorToInt(playerPosition.x / tileSize),
            Mathf.FloorToInt(playerPosition.z / tileSize)
        );
        
        // 加载周围的区块
        for (int x = -1; x <= 1; x++)
        {
            for (int y = -1; y <= 1; y++)
            {
                Vector2Int tileCoord = playerTile + new Vector2Int(x, y);
                LoadTile(tileCoord);
            }
        }
        
        // 卸载远处的区块
        List<Vector2Int> tilesToRemove = new List<Vector2Int>();
        foreach (var tile in tiles.Keys)
        {
            if (Mathf.Abs(tile.x - playerTile.x) > 1 ||
                Mathf.Abs(tile.y - playerTile.y) > 1)
            {
                tilesToRemove.Add(tile);
            }
        }
        
        foreach (var tile in tilesToRemove)
        {
            UnloadTile(tile);
        }
    }
}
```

## 7. 实战案例

### 7.1 巡逻系统

```csharp
public class PatrolSystem : MonoBehaviour
{
    private NavMeshAgent agent;
    public Transform[] patrolPoints;
    private int currentPointIndex = 0;
    
    public float waitTime = 2f;
    private float waitTimer = 0f;
    private bool isWaiting = false;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        if (patrolPoints.Length > 0)
        {
            SetNextDestination();
        }
    }
    
    void Update()
    {
        if (isWaiting)
        {
            waitTimer += Time.deltaTime;
            if (waitTimer >= waitTime)
            {
                isWaiting = false;
                SetNextDestination();
            }
            return;
        }
        
        if (!agent.pathPending && agent.remainingDistance <= agent.stoppingDistance)
        {
            isWaiting = true;
            waitTimer = 0f;
        }
    }
    
    void SetNextDestination()
    {
        if (patrolPoints.Length == 0) return;
        
        agent.SetDestination(patrolPoints[currentPointIndex].position);
        currentPointIndex = (currentPointIndex + 1) % patrolPoints.Length;
    }
}
```

### 7.2 追踪系统

```csharp
public class ChaseSystem : MonoBehaviour
{
    private NavMeshAgent agent;
    public Transform target;
    
    public float detectionRange = 10f;
    public float loseTargetRange = 15f;
    public float updateRate = 0.2f;
    
    private bool isChasing = false;
    private Vector3 lastKnownPosition;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        StartCoroutine(UpdatePath());
    }
    
    IEnumerator UpdatePath()
    {
        while (true)
        {
            if (target != null)
            {
                float distanceToTarget = Vector3.Distance(transform.position, target.position);
                
                if (distanceToTarget <= detectionRange)
                {
                    // 发现目标
                    isChasing = true;
                    lastKnownPosition = target.position;
                    agent.SetDestination(lastKnownPosition);
                }
                else if (isChasing && distanceToTarget > loseTargetRange)
                {
                    // 丢失目标
                    isChasing = false;
                    agent.ResetPath();
                }
                else if (isChasing)
                {
                    // 继续追踪
                    agent.SetDestination(target.position);
                }
            }
            
            yield return new WaitForSeconds(updateRate);
        }
    }
    
    void OnDrawGizmos()
    {
        // 绘制检测范围
        Gizmos.color = Color.yellow;
        Gizmos.DrawWireSphere(transform.position, detectionRange);
        
        Gizmos.color = Color.red;
        Gizmos.DrawWireSphere(transform.position, loseTargetRange);
    }
}
```

### 7.3 群体AI系统

```csharp
public class FlockingSystem : MonoBehaviour
{
    public List<NavMeshAgent> agents = new List<NavMeshAgent>();
    
    [Header("Flocking Parameters")]
    public float cohesionWeight = 1f;
    public float separationWeight = 1f;
    public float alignmentWeight = 1f;
    public float separationRadius = 2f;
    public float neighborRadius = 5f;
    
    void Update()
    {
        foreach (var agent in agents)
        {
            if (agent == null) continue;
            
            Vector3 cohesion = CalculateCohesion(agent);
            Vector3 separation = CalculateSeparation(agent);
            Vector3 alignment = CalculateAlignment(agent);
            
            Vector3 movement = cohesion * cohesionWeight +
                             separation * separationWeight +
                             alignment * alignmentWeight;
            
            // 应用移动
            if (movement.magnitude > 0.1f)
            {
                agent.SetDestination(agent.transform.position + movement.normalized);
            }
        }
    }
    
    Vector3 CalculateCohesion(NavMeshAgent agent)
    {
        Vector3 centerOfMass = Vector3.zero;
        int count = 0;
        
        foreach (var other in agents)
        {
            if (other == agent) continue;
            
            float distance = Vector3.Distance(agent.transform.position, other.transform.position);
            if (distance <= neighborRadius)
            {
                centerOfMass += other.transform.position;
                count++;
            }
        }
        
        if (count > 0)
        {
            centerOfMass /= count;
            return centerOfMass - agent.transform.position;
        }
        
        return Vector3.zero;
    }
    
    Vector3 CalculateSeparation(NavMeshAgent agent)
    {
        Vector3 separation = Vector3.zero;
        
        foreach (var other in agents)
        {
            if (other == agent) continue;
            
            float distance = Vector3.Distance(agent.transform.position, other.transform.position);
            if (distance <= separationRadius)
            {
                Vector3 moveAway = agent.transform.position - other.transform.position;
                separation += moveAway.normalized / distance;
            }
        }
        
        return separation;
    }
    
    Vector3 CalculateAlignment(NavMeshAgent agent)
    {
        Vector3 averageVelocity = Vector3.zero;
        int count = 0;
        
        foreach (var other in agents)
        {
            if (other == agent) continue;
            
            float distance = Vector3.Distance(agent.transform.position, other.transform.position);
            if (distance <= neighborRadius)
            {
                averageVelocity += other.velocity;
                count++;
            }
        }
        
        if (count > 0)
        {
            averageVelocity /= count;
            return averageVelocity;
        }
        
        return Vector3.zero;
    }
}
```

## 8. 性能优化

### 8.1 NavMesh优化

- 合理设置Agent尺寸
- 适当简化导航网格
- 使用NavMesh区域类型优化路径计算
- 避免频繁重建NavMesh

### 8.2 寻路优化

```csharp
public class PathOptimizer : MonoBehaviour
{
    private NavMeshAgent agent;
    private NavMeshPath path;
    
    public float cornerOptimizationDistance = 0.5f;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        path = new NavMeshPath();
    }
    
    // 优化路径拐点
    public Vector3[] OptimizePath(Vector3[] originalPath)
    {
        if (originalPath.Length <= 2) return originalPath;
        
        List<Vector3> optimizedPath = new List<Vector3>();
        optimizedPath.Add(originalPath[0]);
        
        for (int i = 1; i < originalPath.Length - 1; i++)
        {
            Vector3 current = originalPath[i];
            Vector3 next = originalPath[i + 1];
            
            // 检查是否可以跳过当前点
            if (Vector3.Distance(current, next) > cornerOptimizationDistance)
            {
                NavMeshHit hit;
                if (!NavMesh.Raycast(optimizedPath[optimizedPath.Count - 1], next, out hit, NavMesh.AllAreas))
                {
                    continue; // 可以直接连接到下一个点
                }
            }
            
            optimizedPath.Add(current);
        }
        
        optimizedPath.Add(originalPath[originalPath.Length - 1]);
        return optimizedPath.ToArray();
    }
    
    // 使用协程更新路径
    IEnumerator UpdatePathCoroutine()
    {
        while (true)
        {
            if (agent.hasPath)
            {
                Vector3[] currentPath = new Vector3[agent.path.corners.Length];
                agent.path.CopyTo(currentPath, 0);
                
                Vector3[] optimizedPath = OptimizePath(currentPath);
                if (optimizedPath.Length < currentPath.Length)
                {
                    agent.SetPath(path);
                }
            }
            
            yield return new WaitForSeconds(0.5f);
        }
    }
}
```

### 8.3 Agent优化

```csharp
public class AgentOptimizer : MonoBehaviour
{
    private NavMeshAgent agent;
    public float updateInterval = 0.2f;
    public float activationDistance = 20f;
    
    private Transform player;
    private Coroutine updateCoroutine;
    
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        player = GameObject.FindGameObjectWithTag("Player").transform;
        
        StartCoroutine(DistanceCheckCoroutine());
    }
    
    IEnumerator DistanceCheckCoroutine()
    {
        while (true)
        {
            float distance = Vector3.Distance(transform.position, player.position);
            
            if (distance <= activationDistance)
            {
                if (updateCoroutine == null)
                {
                    updateCoroutine = StartCoroutine(UpdatePathCoroutine());
                }
            }
            else
            {
                if (updateCoroutine != null)
                {
                    StopCoroutine(updateCoroutine);
                    updateCoroutine = null;
                    agent.ResetPath();
                }
            }
            
            yield return new WaitForSeconds(0.5f);
        }
    }
    
    IEnumerator UpdatePathCoroutine()
    {
        while (true)
        {
            agent.SetDestination(player.position);
            yield return new WaitForSeconds(updateInterval);
        }
    }
}
```
