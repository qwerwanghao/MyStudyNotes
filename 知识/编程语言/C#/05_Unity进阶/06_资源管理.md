# Unity资源管理系统

## 1. 资源管理概述

Unity资源管理是游戏开发中的关键环节，良好的资源管理可以提高游戏性能、减少内存占用、优化加载时间。主要包括以下几个方面：

- 资源类型管理
- 资源加载策略
- 资源卸载机制
- 内存管理
- 资源打包与分发

### 1.1 资源类型

Unity中的主要资源类型：

- **场景文件**（.scene）
- **预制体**（.prefab）
- **材质**（.mat）
- **纹理**（.png, .jpg, .tga等）
- **网格**（.fbx, .obj等）
- **音频**（.mp3, .wav等）
- **动画**（.anim）
- **脚本**（.cs）
- **着色器**（.shader）

### 1.2 资源管理原则

1. 合理组织资源结构
2. 及时释放不需要的资源
3. 避免重复加载
4. 控制资源粒度
5. 优化资源依赖关系

## 2. Resources系统

### 2.1 基本使用

```csharp
public class ResourceLoader : MonoBehaviour
{
    // 同步加载
    public GameObject LoadPrefab(string prefabName)
    {
        GameObject prefab = Resources.Load<GameObject>("Prefabs/" + prefabName);
        if (prefab != null)
        {
            return Instantiate(prefab);
        }
        Debug.LogError($"Prefab {prefabName} not found!");
        return null;
    }
    
    // 异步加载
    public IEnumerator LoadPrefabAsync(string prefabName, System.Action<GameObject> callback)
    {
        ResourceRequest request = Resources.LoadAsync<GameObject>("Prefabs/" + prefabName);
        yield return request;
        
        if (request.asset != null)
        {
            GameObject instance = Instantiate(request.asset as GameObject);
            callback?.Invoke(instance);
        }
        else
        {
            Debug.LogError($"Prefab {prefabName} not found!");
            callback?.Invoke(null);
        }
    }
    
    // 卸载资源
    public void UnloadUnusedResources()
    {
        Resources.UnloadUnusedAssets();
        System.GC.Collect();
    }
}
```

### 2.2 Resources的局限性

- 无法动态更新
- 所有资源打包进安装包
- 资源加载类型固定
- 不支持细粒度管理

### 3.2 AssetBundle加载

```csharp
public class AssetBundleManager : MonoBehaviour
{
    private Dictionary<string, AssetBundle> loadedBundles = new Dictionary<string, AssetBundle>();
    
    // 同步加载
    public AssetBundle LoadBundle(string bundleName)
    {
        if (loadedBundles.ContainsKey(bundleName))
        {
            return loadedBundles[bundleName];
        }
        
        string bundlePath = Path.Combine(Application.streamingAssetsPath, bundleName);
        AssetBundle bundle = AssetBundle.LoadFromFile(bundlePath);
        if (bundle != null)
        {
            loadedBundles.Add(bundleName, bundle);
        }
        return bundle;
    }
    
    // 异步加载
    public IEnumerator LoadBundleAsync(string bundleName, System.Action<AssetBundle> callback)
    {
        if (loadedBundles.ContainsKey(bundleName))
        {
            callback?.Invoke(loadedBundles[bundleName]);
            yield break;
        }
        
        string bundlePath = Path.Combine(Application.streamingAssetsPath, bundleName);
        AssetBundleCreateRequest request = AssetBundle.LoadFromFileAsync(bundlePath);
        yield return request;
        
        if (request.assetBundle != null)
        {
            loadedBundles.Add(bundleName, request.assetBundle);
            callback?.Invoke(request.assetBundle);
        }
        else
        {
            callback?.Invoke(null);
        }
    }
    
    // 卸载Bundle
    public void UnloadBundle(string bundleName, bool unloadAllLoadedObjects = false)
    {
        if (loadedBundles.TryGetValue(bundleName, out AssetBundle bundle))
        {
            bundle.Unload(unloadAllLoadedObjects);
            loadedBundles.Remove(bundleName);
        }
    }
}
```

### 3.3 依赖管理

```csharp
public class AssetBundleDependencyManager : MonoBehaviour
{
    private AssetBundleManifest manifest;
    private Dictionary<string, string[]> dependencyCache = new Dictionary<string, string[]>();
    
    public void LoadManifest(AssetBundle manifestBundle)
    {
        manifest = manifestBundle.LoadAsset<AssetBundleManifest>("AssetBundleManifest");
    }
    
    public string[] GetDependencies(string bundleName)
    {
        if (dependencyCache.ContainsKey(bundleName))
        {
            return dependencyCache[bundleName];
        }
        
        string[] dependencies = manifest.GetAllDependencies(bundleName);
        dependencyCache.Add(bundleName, dependencies);
        return dependencies;
    }
    
    public IEnumerator LoadBundleWithDependencies(string bundleName, System.Action<AssetBundle> callback)
    {
        // 加载依赖
        string[] dependencies = GetDependencies(bundleName);
        foreach (string dependency in dependencies)
        {
            yield return LoadBundleAsync(dependency, null);
        }
        
        // 加载主Bundle
        yield return LoadBundleAsync(bundleName, callback);
    }
}
```

### 4.3 资源组管理

```csharp
public class AddressableGroupManager : MonoBehaviour
{
    private Dictionary<string, List<AsyncOperationHandle>> groupOperations = 
        new Dictionary<string, List<AsyncOperationHandle>>();
    
    // 加载资源组
    public async Task LoadAssetGroup(string groupName)
    {
        try
        {
            var loadOperation = Addressables.DownloadDependenciesAsync(groupName);
            await loadOperation.Task;
            
            if (!groupOperations.ContainsKey(groupName))
            {
                groupOperations[groupName] = new List<AsyncOperationHandle>();
            }
            groupOperations[groupName].Add(loadOperation);
        }
        catch (Exception e)
        {
            Debug.LogError($"Failed to load asset group {groupName}: {e.Message}");
        }
    }
    
    // 释放资源组
    public void ReleaseAssetGroup(string groupName)
    {
        if (groupOperations.TryGetValue(groupName, out var operations))
        {
            foreach (var operation in operations)
            {
                Addressables.Release(operation);
            }
            groupOperations.Remove(groupName);
        }
    }
}
```

## 5. 内存管理

### 5.1 内存监控

```csharp
public class MemoryMonitor : MonoBehaviour
{
    public float memoryThreshold = 800f; // MB
    public float checkInterval = 5f; // 秒
    
    private void Start()
    {
        StartCoroutine(MonitorMemoryUsage());
    }
    
    IEnumerator MonitorMemoryUsage()
    {
        while (true)
        {
            float totalMemory = Profiler.GetTotalAllocatedMemoryLong() / (1024f * 1024f); // 转换为MB
            
            if (totalMemory > memoryThreshold)
            {
                Debug.LogWarning($"Memory usage ({totalMemory:F2}MB) exceeds threshold!");
                CleanupMemory();
            }
            
            yield return new WaitForSeconds(checkInterval);
        }
    }
    
    void CleanupMemory()
    {
        Resources.UnloadUnusedAssets();
        System.GC.Collect();
        Debug.Log("Memory cleanup performed");
    }
}
```

### 5.2 资源池化

```csharp
public class ObjectPool : MonoBehaviour
{
    [System.Serializable]
    public class Pool
    {
        public string tag;
        public GameObject prefab;
        public int size;
    }
    
    public List<Pool> pools;
    private Dictionary<string, Queue<GameObject>> poolDictionary;
    
    void Start()
    {
        poolDictionary = new Dictionary<string, Queue<GameObject>>();
        
        foreach (Pool pool in pools)
        {
            Queue<GameObject> objectPool = new Queue<GameObject>();
            
            for (int i = 0; i < pool.size; i++)
            {
                GameObject obj = Instantiate(pool.prefab);
                obj.SetActive(false);
                objectPool.Enqueue(obj);
            }
            
            poolDictionary.Add(pool.tag, objectPool);
        }
    }
    
    public GameObject SpawnFromPool(string tag, Vector3 position, Quaternion rotation)
    {
        if (!poolDictionary.ContainsKey(tag))
        {
            Debug.LogWarning($"Pool with tag {tag} doesn't exist.");
            return null;
        }
        
        Queue<GameObject> pool = poolDictionary[tag];
        
        if (pool.Count == 0)
        {
            Debug.LogWarning($"Pool with tag {tag} is empty.");
            return null;
        }
        
        GameObject objectToSpawn = pool.Dequeue();
        objectToSpawn.SetActive(true);
        objectToSpawn.transform.position = position;
        objectToSpawn.transform.rotation = rotation;
        
        pool.Enqueue(objectToSpawn);
        
        return objectToSpawn;
    }
}
```

## 6. 资源热更新

### 6.1 版本控制

```csharp
public class VersionManager : MonoBehaviour
{
    [System.Serializable]
    public class VersionInfo
    {
        public string version;
        public string[] updatedBundles;
        public long totalSize;
    }
    
    private string versionUrl = "http://your-server.com/version.json";
    private string currentVersion;
    
    public async Task<VersionInfo> CheckForUpdates()
    {
        try
        {
            using (UnityWebRequest www = UnityWebRequest.Get(versionUrl))
            {
                await www.SendWebRequest();
                
                if (www.result == UnityWebRequest.Result.Success)
                {
                    VersionInfo serverVersion = JsonUtility.FromJson<VersionInfo>(www.downloadHandler.text);
                    if (serverVersion.version != currentVersion)
                    {
                        return serverVersion;
                    }
                }
            }
        }
        catch (Exception e)
        {
            Debug.LogError($"Failed to check for updates: {e.Message}");
        }
        
        return null;
    }
}
```

### 6.2 资源下载

```csharp
public class AssetDownloader : MonoBehaviour
{
    private string downloadUrl = "http://your-server.com/assets/";
    
    public async Task<bool> DownloadAssetBundle(string bundleName, System.Action<float> progressCallback)
    {
        string url = downloadUrl + bundleName;
        string savePath = Path.Combine(Application.persistentDataPath, bundleName);
        
        try
        {
            using (UnityWebRequest www = UnityWebRequest.Get(url))
            {
                www.downloadHandler = new DownloadHandlerFile(savePath);
                
                UnityWebRequestAsyncOperation operation = www.SendWebRequest();
                
                while (!operation.isDone)
                {
                    progressCallback?.Invoke(www.downloadProgress);
                    await Task.Yield();
                }
                
                if (www.result == UnityWebRequest.Result.Success)
                {
                    return true;
                }
                else
                {
                    Debug.LogError($"Failed to download {bundleName}: {www.error}");
                    return false;
                }
            }
        }
        catch (Exception e)
        {
            Debug.LogError($"Download error: {e.Message}");
            return false;
        }
    }
}
```

### 6.3 增量更新

```csharp
public class IncrementalUpdater : MonoBehaviour
{
    [System.Serializable]
    public class BundleInfo
    {
        public string name;
        public string hash;
        public long size;
    }
    
    private Dictionary<string, BundleInfo> localBundles = new Dictionary<string, BundleInfo>();
    
    public async Task<List<BundleInfo>> GetUpdateList(List<BundleInfo> serverBundles)
    {
        List<BundleInfo> updateList = new List<BundleInfo>();
        
        foreach (var serverBundle in serverBundles)
        {
            if (!localBundles.ContainsKey(serverBundle.name) ||
                localBundles[serverBundle.name].hash != serverBundle.hash)
            {
                updateList.Add(serverBundle);
            }
        }
        
        return updateList;
    }
    
    public async Task UpdateBundles(List<BundleInfo> updateList, System.Action<float> progressCallback)
    {
        int totalBundles = updateList.Count;
        int completedBundles = 0;
        
        foreach (var bundle in updateList)
        {
            bool success = await DownloadAndVerifyBundle(bundle);
            if (success)
            {
                completedBundles++;
                progressCallback?.Invoke((float)completedBundles / totalBundles);
            }
        }
    }
    
    private async Task<bool> DownloadAndVerifyBundle(BundleInfo bundle)
    {
        AssetDownloader downloader = new AssetDownloader();
        bool success = await downloader.DownloadAssetBundle(bundle.name, null);
        
        if (success)
        {
            // 验证文件完整性
            string filePath = Path.Combine(Application.persistentDataPath, bundle.name);
            string fileHash = CalculateFileHash(filePath);
            
            if (fileHash == bundle.hash)
            {
                localBundles[bundle.name] = bundle;
                return true;
            }
        }
        
        return false;
    }
    
    private string CalculateFileHash(string filePath)
    {
        using (var md5 = System.Security.Cryptography.MD5.Create())
        using (var stream = File.OpenRead(filePath))
        {
            byte[] hash = md5.ComputeHash(stream);
            return BitConverter.ToString(hash).Replace("-", "").ToLowerInvariant();
        }
    }
}
```

## 7. 最佳实践

### 7.1 资源命名规范

- 使用有意义的名称
- 遵循一致的命名约定
- 避免特殊字符
- 使用前缀或后缀标识资源类型
