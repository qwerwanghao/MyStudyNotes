# 类和对象

## 类的基本概念

### 类的定义
```csharp
public class Player
{
    // 字段
    private int health;
    private string playerName;
    
    // 构造函数
    public Player(string name)
    {
        playerName = name;
        health = 100;
    }
    
    // 方法
    public void TakeDamage(int damage)
    {
        health -= damage;
        if (health < 0) health = 0;
    }
}
```

### 访问修饰符
1. **public**
   - 可以被任何代码访问
   - 适用于对外公开的接口

2. **private**
   - 只能在当前类中访问
   - 用于实现封装

3. **protected**
   - 可以在当前类和派生类中访问
   - 用于继承场景

4. **internal**
   - 同一程序集中的任何代码都可访问
   - 用于程序集级别的访问控制

## 类的成员

### 字段
```csharp
public class Enemy
{
    // 常量字段
    public const int MAX_HEALTH = 100;
    
    // 只读字段
    public readonly string enemyId;
    
    // 私有字段
    private int health;
    private float speed;
    
    // 静态字段
    public static int enemyCount;
}
```

### 属性
```csharp
public class Character
{
    private int health;
    
    // 自动属性
    public string Name { get; set; }
    
    // 带有逻辑的属性
    public int Health
    {
        get { return health; }
        set 
        {
            health = value;
            if (health < 0) health = 0;
            OnHealthChanged?.Invoke(health);
        }
    }
    
    // 只读属性
    public bool IsAlive => health > 0;
}
```

### 方法
```csharp
public class Weapon
{
    // 实例方法
    public void Fire()
    {
        if (CanFire())
        {
            PerformFire();
        }
    }
    
    // 私有辅助方法
    private bool CanFire()
    {
        return ammo > 0 && !isReloading;
    }
    
    // 静态方法
    public static Weapon CreateDefaultWeapon()
    {
        return new Weapon { ammo = 30 };
    }
}
```

## 对象的创建和使用

### 对象实例化
```csharp
// 使用new关键字
Player player1 = new Player("Player1");

// 使用var关键字（类型推断）
var player2 = new Player("Player2");

// 对象初始化器
Enemy enemy = new Enemy 
{
    Name = "Boss",
    Health = 1000,
    Speed = 5f
};
```

### 对象引用
```csharp
Player playerA = new Player("A");
Player playerB = playerA;  // 引用同一个对象

// 引用判断
if (playerA == playerB)    // true
if (ReferenceEquals(playerA, playerB))  // true

// null检查
if (playerA != null)
{
    playerA.TakeDamage(10);
}

// null条件运算符
playerA?.TakeDamage(10);
```

## 封装和信息隐藏

### 封装原则
```csharp
public class Bank
{
    private decimal balance;
    
    public decimal Balance
    {
        get { return balance; }
        private set { balance = value; }
    }
    
    public bool Withdraw(decimal amount)
    {
        if (amount <= 0 || amount > balance)
            return false;
            
        balance -= amount;
        return true;
    }
    
    public void Deposit(decimal amount)
    {
        if (amount > 0)
        {
            balance += amount;
        }
    }
}
```

### 访问控制
```csharp
public class GameManager
{
    // 私有构造函数
    private GameManager() { }
    
    // 私有静态实例
    private static GameManager instance;
    
    // 公共访问点
    public static GameManager Instance
    {
        get
        {
            if (instance == null)
            {
                instance = new GameManager();
            }
            return instance;
        }
    }
}
```

## 最佳实践

### 命名约定
```csharp
public class PlayerCharacter
{
    // 私有字段使用camelCase
    private int healthPoints;
    
    // 公共属性使用PascalCase
    public int HealthPoints
    {
        get { return healthPoints; }
        set { healthPoints = value; }
    }
    
    // 方法使用PascalCase
    public void TakeDamage(int amount)
    {
        // 局部变量使用camelCase
        int actualDamage = CalculateDamage(amount);
        healthPoints -= actualDamage;
    }
}
```

### 设计原则
1. **单一职责原则**
   ```csharp
   // 好的设计
   public class PlayerInput { }
   public class PlayerMovement { }
   public class PlayerHealth { }
   
   // 避免
   public class Player 
   {
       // 处理所有功能
   }
   ```

2. **依赖注入**
   ```csharp
   public class WeaponSystem
   {
       private IAmmoSystem ammoSystem;
       
       public WeaponSystem(IAmmoSystem ammoSystem)
       {
           this.ammoSystem = ammoSystem;
       }
   }
