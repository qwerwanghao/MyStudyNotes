# 编程语言学习笔记

> 系统性的编程语言学习记录

## 语言特点对比

### C语言
- 底层语言,直接操作内存
- 运行效率高
- 主要用于系统编程
- 学习重点:指针、内存管理

### C#语言
- 面向对象
- 强类型安全
- 跨平台(.NET Core)
- Unity游戏开发首选语言
- 学习重点:面向对象、委托事件

## 学习路线建议

1. C语言基础(推荐先学)
   - 基础语法
   - 指针与内存
   - 编译原理

2. C#语言进阶
   - 面向对象基础
   - .NET框架
   - Unity开发
