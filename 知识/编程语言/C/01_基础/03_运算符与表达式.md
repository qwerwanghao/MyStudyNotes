# C语言运算符与表达式

## 运算符优先级表
| 优先级 | 运算符 | 结合性 | 描述 |
|--------|---------|--------|------|
| 1 | () [] -> . | 从左到右 | 函数调用、数组下标、成员访问 |
| 2 | ! ~ ++ -- + - * & sizeof | 从右到左 | 一元运算符 |
| 3 | * / % | 从左到右 | 算术运算符 |
| 4 | + - | 从左到右 | 算术运算符 |
| 5 | << >> | 从左到右 | 位移运算符 |
| 6 | < <= > >= | 从左到右 | 关系运算符 |
| 7 | == != | 从左到右 | 相等运算符 |
| 8-13 | &, ^, \|, &&, \|\|, ?: | 从左到右 | 位运算、逻辑运算、条件运算 |
| 14 | = += -= *= /= %= &= ^= \|= <<= >>= | 从右到左 | 赋值运算符 |
| 15 | , | 从左到右 | 逗号运算符 |

## 算术运算符
```c
// 基本运算
int a = 10, b = 3;
int sum = a + b;     // 加法
int diff = a - b;    // 减法
int prod = a * b;    // 乘法
int quot = a / b;    // 整除（3）
int rem = a % b;     // 取余（1）
float div = (float)a / b;  // 浮点除法（3.333...）

// 自增自减
int x = 5;
x++;    // 后增（先使用后增加）
++x;    // 前增（先增加后使用）
```

## 位运算技巧
```c
// 常用位运算技巧
int n = 5;          // 二进制：101
int isEven = !(n & 1);  // 判断奇偶
int isPowerOf2 = n && !(n & (n-1));  // 判断2的幂

// 位操作示例
unsigned int setBit = n | (1 << 3);   // 设置第3位为1
unsigned int clearBit = n & ~(1 << 3);// 清除第3位
unsigned int toggleBit = n ^ (1 << 3);// 反转第3位
```

## 逻辑运算符
```c
// 短路求值
if (ptr != NULL && ptr->value > 0) {
    // 安全的指针访问
}

// 避免常见错误
if (a = b) {     // 错误：赋值而非比较
    // ...
}

if (a == b) {    // 正确：比较
    // ...
}
```

## 运算符陷阱

### 1. 整数溢出
```c
int max = INT_MAX;
max++;   // 溢出：变为 INT_MIN
```

### 2. 除零错误
```c
int safeDivision(int a, int b) {
    if (b == 0) {
        fprintf(stderr, "除数不能为零\n");
        return 0;
    }
    return a / b;
}
```

### 3. 运算符优先级
```c
// 使用括号明确优先级
int result = (a + b) * c;    // 清晰
int mask = 1 << (n & 0x1f);  // 位运算优先级
```

## 条件运算符
```c
// 三元运算符
int max = (a > b) ? a : b;

// 嵌套使用（不推荐）
int value = (a > b) ? 
             (a > c ? a : c) : 
             (b > c ? b : c);
```

## 逗号运算符
```c
// 多重赋值
for (i = 0, j = n-1; i < j; i++, j--) {
    // 回文检查
}

// 返回最后一个表达式的值
int x = (a++, b++, c++);  // x获得c++的值
```

## 最佳实践

### 1. 运算符间添加空格
```c
sum = a + b * c;    // 好
sum=a+b*c;         // 不好
```

### 2. 使用括号明确优先级
```c
result = (a + b) * (c + d);  // 清晰
result = a + b * c + d;      // 可能混淆
```

### 3. 避免副作用
```c
// 不要在同一表达式中多次修改同一变量
x = x++;    // 未定义行为
i = ++i + i++;  // 避免这样写
```
