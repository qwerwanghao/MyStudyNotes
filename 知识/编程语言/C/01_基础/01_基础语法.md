# C语言基础语法

## 程序结构
```c
#include <stdio.h>  // 头文件包含

int main() {        // 主函数
    // 语句
    return 0;       // 返回值
}
```

## 标识符规则
- **必须以字母或下划线开头**
- **区分大小写**
- **不能使用关键字**

### 命名约定
```c
int user_age;      // 下划线分隔（推荐）
int userId;        // 驼峰命名
#define MAX_SIZE 100  // 常量全大写
```

## 关键字
| 类别 | 关键字 |
|------|---------|
| 数据类型 | void, int, char, float, double, ... |
| 控制流 | if, else, for, while, do, switch, ... |
| 存储类 | auto, static, extern, register, ... |
| 其他 | return, break, continue, sizeof, ... |

## 注释规范
```c
// 单行注释

/*
 * 多行注释
 * 建议每行前加*对齐
 */

/** 
 * 文档注释
 * @param x 参数说明
 * @return 返回值说明
 */
```

## 代码风格
```c
// 良好的缩进和间距
if (condition) {
    statement1;
    statement2;
} else {
    statement3;
}

// 运算符两边加空格
sum = a + b * c;

// 函数调用无空格
printf("Hello");
```

## 常见错误
1. **语法错误**
```c
if (x == 5)        // 正确
if (x = 5)         // 错误：赋值而非比较
```

2. **分号使用**
```c
int x = 5;         // 正确
int x = 5          // 错误：缺少分号
```

## 编码规范建议
1. **使用有意义的变量名**
```c
int n;             // 不好
int count;         // 好
```

2. **适当使用空行分隔逻辑块**
```c
void function() {
    // 初始化部分
    int x = 0;
    int y = 0;
    
    // 计算部分
    x = compute(y);
    
    // 输出部分
    printf("%d\n", x);
}
```

3. **括号对齐**
```c
if (condition) {
    // 正确的对齐
}

if (condition) 
    {
    // 不推荐的对齐
    }
```

## 最佳实践
1. **头文件保护**
```c
#ifndef HEADER_H
#define HEADER_H
// 头文件内容
#endif
```

2. **常量定义**
```c
#define PI 3.14159     // 宏定义
const double pi = 3.14159;  // const常量
```

3. **错误处理**
```c
if (ptr == NULL) {
    fprintf(stderr, "Error: Memory allocation failed\n");
    return -1;
}
```

---

> 提示：良好的编码风格有助于提高代码可读性和维护性。
