# C语言文件操作

## 文件基础

### 文件打开模式
| 模式 | 描述 | 说明 |
|------|------|------|
| "r" | 只读 | 文件必须存在 |
| "w" | 只写 | 创建新文件，覆盖已有文件 |
| "a" | 追加 | 在文件末尾追加内容 |
| "r+" | 读写 | 文件必须存在 |
| "w+" | 读写 | 创建新文件，覆盖已有文件 |
| "a+" | 读写 | 追加或创建新文件 |

### 基本文件操作
```c
// 打开文件
FILE *fp = fopen("test.txt", "r");
if (fp == NULL) {
    perror("文件打开失败");
    return -1;
}

// 关闭文件
fclose(fp);
```

## 文件读写操作

### 字符操作
```c
// 读取单个字符
int ch = fgetc(fp);
if (ch == EOF) {
    // 处理文件结束或错误
}

// 写入单个字符
fputc('A', fp);
```

### 字符串操作
```c
char line[256];

// 读取一行
if (fgets(line, sizeof(line), fp) != NULL) {
    // 处理读取的字符串
}

// 写入字符串
fputs("Hello, World!\n", fp);
```

### 格式化读写
```c
// 格式化读取
int age;
char name[50];
fscanf(fp, "%d %s", &age, name);

// 格式化写入
fprintf(fp, "Age: %d, Name: %s\n", age, name);
```

## 二进制文件操作

### 块读写
```c
struct Record {
    int id;
    char name[50];
};

// 写入结构体
struct Record record = {1, "John"};
fwrite(&record, sizeof(struct Record), 1, fp);

// 读取结构体
struct Record readRecord;
size_t count = fread(&readRecord, sizeof(struct Record), 1, fp);
if (count < 1) {
    // 处理读取错误
}
```

## 文件定位

### 文件指针操作
```c
// 获取当前位置
long pos = ftell(fp);

// 设置文件指针位置
fseek(fp, 0, SEEK_SET);  // 文件开头
fseek(fp, 0, SEEK_END);  // 文件末尾
fseek(fp, offset, SEEK_CUR);  // 相对当前位置

// 重置文件指针
rewind(fp);  // 等同于 fseek(fp, 0, SEEK_SET)
```

## 文件操作最佳实践

### 1. 错误处理
```c
FILE *safeOpen(const char *filename, const char *mode) {
    FILE *fp = fopen(filename, mode);
    if (fp == NULL) {
        fprintf(stderr, "无法打开文件 %s: %s\n", 
                filename, strerror(errno));
        return NULL;
    }
    return fp;
}
```

### 2. 资源管理
```c
// 使用宏确保文件关闭
#define CLOSE_FILE(fp) do { \
    if (fp) { \
        fclose(fp); \
        fp = NULL; \
    } \
} while(0)

FILE *fp = fopen("test.txt", "r");
if (!fp) return -1;

// 处理文件...

CLOSE_FILE(fp);
```

### 3. 缓冲区管理
```c
// 设置缓冲区
char buffer[BUFSIZ];
setvbuf(fp, buffer, _IOFBF, sizeof(buffer));

// 强制刷新缓冲区
fflush(fp);
```

## 高级文件操作

### 1. 临时文件
```c
// 创建临时文件
FILE *temp = tmpfile();
if (temp == NULL) {
    perror("临时文件创建失败");
    return -1;
}
```

### 2. 文件锁定
```c
#include <fcntl.h>
#include <unistd.h>

// 设置文件锁
struct flock fl = {
    .l_type = F_WRLCK,
    .l_whence = SEEK_SET,
    .l_start = 0,
    .l_len = 0
};

int fd = fileno(fp);
if (fcntl(fd, F_SETLK, &fl) == -1) {
    perror("文件锁定失败");
}
```

## 性能优化

### 1. 缓冲区大小
```c
// 使用合适的缓冲区大小
#define BUFFER_SIZE 4096
char buffer[BUFFER_SIZE];

size_t bytesRead;
while ((bytesRead = fread(buffer, 1, BUFFER_SIZE, fp)) > 0) {
    // 处理数据
}
```

### 2. 批量操作
```c
// 批量读写而不是逐字节操作
#define RECORDS_PER_READ 100
struct Record records[RECORDS_PER_READ];

fread(records, sizeof(struct Record), 
      RECORDS_PER_READ, fp);
```
