# C语言结构体与联合

## 结构体基础

### 结构体定义
```c
struct Person {
    char name[50];
    int age;
    float height;
};

// 使用typedef
typedef struct {
    int x;
    int y;
} Point;
```

### 结构体初始化
```c
// 常规初始化
struct Person person1 = {"John", 25, 175.5};

// 指定初始化器（C99）
struct Person person2 = {
    .name = "Alice",
    .age = 30,
    .height = 165.0
};
```

## 结构体操作

### 成员访问
```c
struct Person person;
strcpy(person.name, "Bob");  // 点运算符
person.age = 20;

struct Person *ptr = &person;
ptr->height = 180.0;        // 箭头运算符
```

### 结构体数组
```c
struct Person team[10];
team[0].age = 25;

// 动态分配
struct Person *group = malloc(n * sizeof(struct Person));
if (group) {
    group[0].age = 30;
    // ...
    free(group);
}
```

## 联合体

### 基本用法
```c
union Data {
    int i;
    float f;
    char str[20];
};

union Data data;
data.i = 10;    // 只能使用一个成员
printf("%d", data.i);
```

### 应用场景
```c
// 节省内存的数据结构
struct Value {
    enum { INT, FLOAT, STRING } type;
    union {
        int i;
        float f;
        char *s;
    } data;
};
```

## 位域

### 定义与使用
```c
struct Flags {
    unsigned int valid : 1;    // 1位
    unsigned int ready : 1;    // 1位
    unsigned int error : 2;    // 2位
};

struct Flags status = {1, 1, 0};
if (status.valid) {
    // 处理
}
```

## 嵌套结构

### 结构体嵌套
```c
struct Address {
    char street[50];
    char city[30];
};

struct Employee {
    char name[50];
    struct Address addr;  // 嵌套结构
    float salary;
};
```

### 自引用结构
```c
struct Node {
    int data;
    struct Node *next;  // 自引用
};
```

## 内存对齐

### 对齐规则
```c
struct Aligned {
    char c;      // 1字节
    int i;       // 4字节
    short s;     // 2字节
};

// 使用紧凑布局
#pragma pack(1)
struct Packed {
    char c;
    int i;
    short s;
};
#pragma pack()
```

## 最佳实践

### 1. 封装性
```c
typedef struct {
    // 私有数据
    char *data;
    size_t size;
} Vector;

// 接口函数
Vector* vector_create(size_t initial_size);
void vector_destroy(Vector *v);
```

### 2. 错误处理
```c
struct Result {
    bool success;
    union {
        int value;      // 成功时的值
        char *error;    // 错误信息
    } data;
};

struct Result compute() {
    struct Result result = {0};
    if (/* error condition */) {
        result.success = false;
        result.data.error = "计算失败";
    }
    return result;
}
```

### 3. 深浅拷贝
```c
// 浅拷贝
struct Person p1 = p2;  // 直接赋值

// 深拷贝
void person_copy(struct Person *dst, const struct Person *src) {
    strcpy(dst->name, src->name);
    dst->age = src->age;
    // 复制其他成员
}
```

## 性能考虑

### 1. 成员排序
```c
// 不好的排序（有间隙）
struct BadLayout {
    char a;     // 1字节
    int b;      // 4字节
    char c;     // 1字节
};

// 好的排序（最小间隙）
struct GoodLayout {
    int b;      // 4字节
    char a;     // 1字节
    char c;     // 1字节
};
```

### 2. 缓存友好
```c
// 数组的结构体 vs 结构体的数组
struct Point points[1000];          // 更好
struct Array {
    int x[1000];
    int y[1000];
} points_array;                     // 较差
```
