# 学习笔记： 全局变量 sizeof

## 一、全局变量

1. **定义**:
   - 全局变量是在函数外部定义的变量，它可以在程序的任何地方被访问。

2. **全局变量的特点**:
   - **内存存在时间**: 一旦全局变量分配内存，它将一直存在，直到程序退出。
   - **内存分配**: 内存是在程序开始运行时就已经分配好的。
   - **全局访问**: 全局变量可以被全局访问。
   - **编译时可见**: 只要在编译时能够找到这个全局变量，就可以使用它。
   - **初始值**: 如果未给定初始值，全局变量的初始值通常为 0。

   ```c
   int global_value = 100; // 定义一个整型的全局变量
   ```

3. **静态全局变量**:
   - 当全局变量被 `static` 修饰时，它的作用范围被限制在声明它的文件中。
   - 静态全局变量只允许在当前的 `.c` 文件内部使用，其他文件无法访问。
   - 使用 `static` 修饰的全局变量可以减少维护成本，适合只在模块内部使用的情况。

   ```c
   static int global_var = 10; // 静态全局变量
   ```

4. **静态局部变量**:
   - 在函数内部定义的静态局部变量只能在该函数内部使用，其他函数无法访问。
   - 函数的调用和返回不会导致静态局部变量的内存分配和释放。

   ```c
   void test_static_var() {
       static int index = 1; // 静态局部变量
       index++;
       printf("index: %d\n", index);
   }
   ```

---

## 二、`sizeof` 运算

1. **计算变量所占内存**:
   - 使用 `sizeof` 运算符计算变量所占的内存大小。

   ```c
   int var; 
   printf("%zu\n", sizeof(var)); // 打印变量 var 所占的内存大小
   ```

2. **计算数据类型所占的内存**:
   - 使用 `sizeof` 运算符计算数据类型所占的内存大小。

   ```c
   printf("%zu\n", sizeof(int)); // 打印 int 类型所占的内存大小
   ```

## 三、一维数组

1. **数组定义**:
   - 数组是用来存储同一类型数据的集合。
   - 数组的定义如下：
     ```c
     int arr[10]; // 定义一个大小为 10 的 int 类型数组
     ```

2. **数组的索引与访问**:
   - 访问并设置数组的元素：
     ```c
     arr[0] = 1; // 设置数组的第一个元素为 1
     ```

3. **使用 `for` 循环遍历数组**:
   - 遍历数组并打印每个元素：
     ```c
     for (int i = 0; i < 10; i++) {
         printf("%d ", arr[i]); // 打印数组的每个元素
     }
     ```

4. **注意事项**:
   - 数组使用时要避免越界：
     - 数组一旦定义，大小就不能改变。
     - 元素个数一旦确定，只能访问索引范围 `[0, 数组大小 - 1]`。
     - 访问越界会导致程序异常。

5. **数组的内存分布**:
   - 数组的内存是连续分布的。
   - 数组的名字实际上是指向数组第一个元素的指针。
   - 例如，`int arr[10];` 会为数组分配连续的内存空间，大小为 10 个 `int` 类型的大小。
   - 访问数组元素 `arr[3]` 时，实际计算的是偏移量：
     - 偏移量 = 索引 × 元素大小。
     - 将偏移量加到数组起始地址上，得到 `arr[3]` 的内存地址。

---

## 四、二维数组

1. **二维数组定义**:
   - 二维数组的内存分布也是连续的，可以看作是一个数组的数组。

2. **二维数组初始化**:
   - 初始化一个 2x3 的二维数组如下：
     ```c
     int arr[2][3] = {{1, 2, 3}, {4, 5, 6}}; // 定义并初始化二维数组
     ```

3. **二维数组的访问与遍历**:
   - 使用嵌套 `for` 循环遍历二维数组并打印每个元素：
     ```c
     for (int i = 0; i < 2; i++) {
         for (int j = 0; j < 3; j++) {
             printf("%d ", arr[i][j]); // 打印二维数组的每个元素
         }
         printf("\n"); // 换行
     }
     ```
