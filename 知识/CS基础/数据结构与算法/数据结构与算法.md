## 七段式课程蓝图

<a id="phase-1"></a>
### 1. 复杂度与分析方法
- 核心问题：为什么同样能跑通的代码，性能差异会很大
- 应覆盖：
  - 时间复杂度
  - 空间复杂度
  - 渐进分析
  - 最坏 / 平均 / 摊还分析

<a id="phase-2"></a>
### 2. 线性结构与基础容器
- 核心问题：最基本的数据该如何组织
- 应覆盖：
  - 数组
  - 链表
  - 栈
  - 队列
  - 双端队列

<a id="phase-3"></a>
### 3. 树、堆、哈希与索引
- 核心问题：如何支持更高效的查找、组织和优先级管理
- 应覆盖：
  - 二叉树
  - 二叉搜索树
  - 平衡树
  - 堆 / 优先队列
  - 哈希表
  - Trie / 并查集（可作为扩展）

<a id="phase-4"></a>
### 4. 排序、查找与分治
- 核心问题：经典算法范式如何把问题拆小并高效求解
- 应覆盖：
  - 二分查找
  - 快排 / 归并 / 堆排
  - 分治思想
  - 稳定性与适用场景

<a id="phase-5"></a>
### 5. 递归、回溯与搜索
- 核心问题：当问题具有状态空间时，如何系统探索解空间
- 应覆盖：
  - 递归思维
  - DFS / BFS
  - 回溯模板
  - 剪枝思路

<a id="phase-6"></a>
### 6. 动态规划与贪心
- 核心问题：怎样从重复子问题和局部选择中构造全局解
- 应覆盖：
  - DP 状态设计
  - 转移方程
  - 记忆化与表格法
  - 贪心正确性直觉

<a id="phase-7"></a>
### 7. 图模型与综合问题
- 核心问题：如何把现实问题抽象成图，并运用统一算法处理
- 应覆盖：
  - 图的表示
  - 最短路
  - 拓扑排序
  - 最小生成树
  - 连通性与综合建模
