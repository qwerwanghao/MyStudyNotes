## 基础概念

### 进制速查表
| 进制类型 | 基数 | 数字范围 | 前缀表示 | 实例 |
|---------|------|----------|----------|------|
| 二进制   | 2    | 0-1      | 0b/0B    | 0b1101 |
| 八进制   | 8    | 0-7      | 0        | 015 |
| 十进制   | 10   | 0-9      | 无        | 13 |
| 十六进制 | 16   | 0-9,A-F  | 0x/0X    | 0xD |

### 进制的本质
```
十进制 13 的本质：
1 * 10¹ + 3 * 10⁰ = 10 + 3 = 13

二进制 1101 的本质：
1 * 2³ + 1 * 2² + 0 * 2¹ + 1 * 2⁰ = 8 + 4 + 0 + 1 = 13
```

## 进制转换方法

### 1. 十进制转其他进制
**核心方法**：除基取余，逆序排列
```
示例：将十进制13转换为二进制
13 ÷ 2 = 6 余 1  ↘
 6 ÷ 2 = 3 余 0   |
 3 ÷ 2 = 1 余 1   | 逆序读取
 1 ÷ 2 = 0 余 1  ↗
结果：1101
```

### 2. 其他进制转十进制
**核心方法**：按权展开
```
示例：二进制1101转十进制
1101 = 1×2³ + 1×2² + 0×2¹ + 1×2⁰
     = 8 + 4 + 0 + 1
     = 13
```

### 3. 二进制与十六进制快速转换
**技巧**：每4位二进制对应1位十六进制
```
二进制组  十六进制
0000      0
0001      1
...       ...
1111      F

示例：1101 0011
1101 → D
0011 → 3
结果：0xD3
```

## C语言实现

### 1. 使用位运算进行转换
```c
// 十进制转二进制（位运算法）
void decToBin(int n) {
    // 注意：从高位到低位打印
    for(int i = 31; i >= 0; i--) {
        printf("%d", (n >> i) & 1);
    }
    printf("\n");
}

// 二进制字符串转十进制
int binToDec(const char* bin) {
    int result = 0;
    while(*bin) {
        // ⚠️ 注意：需要检查输入合法性
        if(*bin != '0' && *bin != '1') return -1;
        result = result * 2 + (*bin++ - '0');
    }
    return result;
}
```

### 2. 使用标准库函数

#include <stdio.h>

## 实际应用

### 1. 位运算优化
```c
// 使用位运算优化乘除法
int multiply_by_16(int x) {
    return x << 4;    // 等价于 x * 16
}

int divide_by_2(int x) {
    return x >> 1;    // 等价于 x / 2
}
```

### 2. 颜色值转换
```c
// RGB颜色转十六进制
unsigned int rgbToHex(int r, int g, int b) {
    return ((r & 0xff) << 16) | ((g & 0xff) << 8) | (b & 0xff);
}

// 十六进制转RGB
void hexToRGB(unsigned int hex, int* r, int* g, int* b) {
    *r = (hex >> 16) & 0xFF;
    *g = (hex >> 8) & 0xFF;
    *b = hex & 0xFF;
}
```

## 易错点与注意事项

1. **溢出问题**
   - 进行移位操作时注意位数限制
   - 使用unsigned类型处理纯二进制运算

2. **符号位处理**
   - 右移运算符在处理负数时的行为
   - 有符号数和无符号数的转换陷阱

3. **格式化输出注意事项**
   ```c
   // 正确的格式化输出
   printf("0x%08X\n", value);  // 补齐8位，大写十六进制
   ```
