# C语言数据类型与变量

## 基本数据类型

### 整数类型
| 类型 | 大小(字节) | 范围 | 格式符 |
|------|------------|------|--------|
| char | 1 | -128 到 127 | %c, %d |
| unsigned char | 1 | 0 到 255 | %u |
| short | 2 | -32768 到 32767 | %hd |
| int | 4 | -2^31 到 2^31-1 | %d |
| long | 4或8 | 依系统而定 | %ld |
| long long | 8 | -2^63 到 2^63-1 | %lld |

### 浮点类型
| 类型 | 大小(字节) | 精度 | 格式符 |
|------|------------|------|--------|
| float | 4 | 6位小数 | %f |
| double | 8 | 15位小数 | %lf |
| long double | 10或16 | 19位小数 | %Lf |

## 变量声明与初始化

### 基本语法
```c
// 声明
type variable_name;

// 声明并初始化
type variable_name = value;

// 多个变量
type var1, var2, var3 = value;
```

### 示例
```c
int count = 0;
float pi = 3.14f;    // f后缀表示float
double price = 9.99;
char grade = 'A';
```

## 类型修饰符

### 存储类
```c
auto int x;      // 默认
register int y;  // 建议存放寄存器
static int z;    // 静态存储
extern int a;    // 外部变量
```

### 类型限定符
```c
const int MAX = 100;      // 常量
volatile int status;      // 易变变量
```

## 类型转换

### 隐式转换
```c
int i = 10;
float f = i;    // int → float
double d = f;   // float → double
```

### 显式转换
```c
float f = 3.14;
int i = (int)f;    // 强制类型转换

// 避免精度丢失的转换
double price = 9.99;
int cents = (int)(price * 100 + 0.5);
```

## 作用域和生命周期

### 局部变量
```c
void function() {
    int x = 10;    // 局部变量
    {
        int y = 20;  // 块作用域
    }
    // y不可访问
}
```

### 全局变量
```c
int globalVar;    // 全局变量

void function() {
    globalVar = 100;  // 可访问全局变量
}
```

## 常见陷阱

### 1. 整数溢出
```c
unsigned char x = 255;
x++;    // 变为0

int max = INT_MAX;
max++;   // 变为INT_MIN
```

### 2. 浮点精度
```c
float a = 0.1;
float b = 0.2;
float c = 0.3;

if (a + b == c) {    // 可能不相等！
    // ...
}
```

## 最佳实践

### 1. 使用适当的数据类型
```c
// 存储年龄
unsigned char age;    // 而不是 int

// 存储大数据
size_t size;         // 而不是 int
```

### 2. 初始化变量
```c
int count = 0;       // 好
int count;          // 不好

// 多变量初始化
int i = 0, j = 0, k = 0;
```

### 3. 使用有意义的常量
```c
#define SECONDS_PER_DAY (24 * 60 * 60)
const double PI = 3.14159265359;
```

---

> 注意：不同编译器和平台可能有不同的数据类型大小
