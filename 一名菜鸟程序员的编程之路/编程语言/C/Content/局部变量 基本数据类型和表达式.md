
# 学习笔记：局部变量 基本数据类型和表达式

## 一、局部变量

### 1. 变量定义
- **变量**是内存的一块名字代号，每个变量对应一块内存。
  - **内存大小**: 由变量类型决定。
  - **数据类型**: 变量存放的数据类型。

### 2. 变量访问
- 使用变量的名字可以访问对应的内存，从而减少开发难度。

### 3. 定义变量
- 变量定义语句格式：
  ```c
  变量类型  变量名字 [ = 初始化数据 ]; // 每条语句必须以分号结尾
  ```
- 这条语句定义了一个变量，并分配了一块内存，变量的名字用于访问这块内存。

### 4. 局部变量
- **定义**: 变量在函数内部定义，称为局部变量。
- **作用域**: 变量的使用范围限于函数内，函数结束后变量被回收。

### 5. 函数参数变量
- 作为参数定义在函数内，与局部变量一样，参数变量在函数结束后也会被回收。

## 二、变量的基本数据类型关键字

### 1. 有符号整数
- **一个字节**: `char 变量名字;` 范围: [-128, 127]
- **两个字节**: `short 变量名字;` 范围: [-2^15, 2^15 - 1]
- **四个字节**: `int 变量名字;` 范围: [-2^31, 2^31 - 1]
- **八个字节**: 
  - `long long 变量名字;` 范围: [-2^63, 2^63 - 1] (GNU 编译器)
  - `__int64 变量名字;` 范围: [-2^63, 2^63 - 1] (VC 编译器)

### 2. 无符号整数
- **一个字节**: `unsigned char 变量名字;` 范围: [0, 255]
- **两个字节**: `unsigned short 变量名字;` 范围: [0, 2^16 - 1]
- **四个字节**: `unsigned int 变量名字;` 范围: [0, 2^32 - 1]
- **八个字节**: 
  - `unsigned long long 变量名字;` 范围: [0, 2^63 - 1] (GNU 编译器)
  - `unsigned __int64 变量名字;` 范围: [0, 2^63 - 1] (VC 编译器)

### 3. 小数
- **四个字节**: `float 变量名字;`
- **八个字节**: `double 变量名字;`

### 4. 字节表示
一个字节可以表示 256 种不同的状态，具体的表示方式是通过 8 个二进制位的组合来实现。例如：

```
00000000
00000001
00000010
...
11111111
```

## 三、定义函数中的局部变量

### 1. 函数定义
```c
返回值类型 函数名字(参数1, 参数2) {
    代码语句1;
    代码语句2;
    // 定义变量
    类型 变量名字;
}
```

### 2. 返回语句
- 使用 `return 值;` 返回结果。如果没有返回值，使用 `void`。

### 3. 使用返回值
- 可以直接使用返回值或使用变量保存返回值。

### 4. 函数参数
- 参数变量与局部变量一样，可以通过参数名字访问参数的内存；函数返回后参数也无法使用。

## 四、基本表达式

### 1. 赋值表达式
- **赋值**: `变量 = 同类变量/常数值/同类的函数返回值;`

### 2. 运算符
- **算术运算**: `+`, `-`, `*`, `/` 可用于变量与变量或变量与常数之间的运算。

### 3. 括号
- **优先级**: 使用 `()` 改变运算的优先级，括号的嵌套有限。

### 4. 自增自减运算
- **`++` 和 `--`**:
  - `++` 在后面: 先计算，再加 1。
  - `++` 在前面: 先加 1，再参与计算。

### 5. 复合赋值运算
- **示例**:
  ```c
  a += value; // 等同于 a = a + value;
  a -= value; // 等同于 a = a - value;
  a *= value; // 等同于 a = a * value;
  a /= value; // 等同于 a = a / value;
  ```
