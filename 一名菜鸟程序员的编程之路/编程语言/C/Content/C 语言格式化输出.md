## 基本格式说明符
| 类型 | 格式说明符 | 描述 | 示例代码 | 输出结果 |
|------|------------|------|-----------|-----------|
| 整数 | `%d`, `%i` | 十进制整数 | `printf("%d", 42);` | `42` |
| 整数 | `%u` | 无符号整数 | `printf("%u", 42);` | `42` |
| 整数 | `%o` | 八进制 | `printf("%o", 42);` | `52` |
| 整数 | `%x`, `%X` | 十六进制 | `printf("%x %X", 42, 42);` | `2a 2A` |
| 浮点 | `%f` | 小数形式 | `printf("%.2f", 3.14159);` | `3.14` |
| 浮点 | `%e`, `%E` | 科学计数 | `printf("%e", 3.14159);` | `3.141590e+00` |
| 浮点 | `%g`, `%G` | 自动选择 | `printf("%g", 3.14159);` | `3.14159` |
| 字符 | `%c` | 单个字符 | `printf("%c", 'A');` | `A` |
| 字符串 | `%s` | 字符串 | `printf("%s", "Hello");` | `Hello` |
| 指针 | `%p` | 地址值 | `printf("%p", &var);` | `0x7fff...` |

## 格式控制修饰符

### 宽度与对齐
```c
printf("%10d", 42);    // 右对齐：[        42]
printf("%-10d", 42);   // 左对齐：[42        ]
printf("%010d", 42);   // 前导零：[0000000042]
```

### 精度控制
```c
printf("%.2f", 3.14159);     // 小数点后位数：3.14
printf("%8.2f", 3.14159);    // 总宽度8，2位小数：[    3.14]
printf("%08.2f", 3.14159);   // 前导零填充：[00003.14]
```

## 常见格式化技巧

### 1. 表格对齐
```c
printf("%-15s %10d\n", "Name", 123);    // 左对齐文本，右对齐数字
printf("%-15s %10.2f\n", "Price", 9.9); // 对齐小数
```

### 2. 千位分隔
```c
printf("%'d\n", 1234567);    // 输出：1,234,567（部分系统支持）
```

### 3. 特殊字符处理
```c
printf("%%");        // 输出百分号：%
printf("\"Hello\""); // 输出引号："Hello"
```

## 实际应用示例

### 格式化表格输出
```c
#include <stdio.h>

int main() {
    printf("%-20s %10s %8s\n", "Product", "Price", "Qty");
    printf("%-20.20s %10.2f %8d\n", "Apple iPhone 13", 999.99, 5);
    printf("%-20.20s %10.2f %8d\n", "Samsung Galaxy S21", 888.88, 3);
    return 0;
}
```

输出：
```
Product               Price      Qty
Apple iPhone 13      999.99       5
Samsung Galaxy S21   888.88       3
```

### 注意事项
1. 确保格式说明符与参数类型匹配
2. 使用精度控制避免浮点数显示过多小数位
3. 考虑字符串长度限制，避免缓冲区溢出
4. 注意不同系统对某些特性的支持可能不同
