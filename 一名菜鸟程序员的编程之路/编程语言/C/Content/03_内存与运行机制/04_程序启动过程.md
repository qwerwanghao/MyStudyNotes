# 程序启动过程

## 程序的启动过程

程序的启动过程可以分为以下几个步骤：

1. **操作系统创建一个进程**  
   操作系统为程序创建一个新的进程，并分配必要的资源。

2. **加载可执行文件**  
   操作系统将 `.exe` 文件的二进制代码加载到进程的代码段中。代码段是只读的，不能被修改。

3. **找到入口指令**  
   加载器根据程序的头部信息找到代码的入口指令，即程序的起始执行点。

4. **执行指令**  
   程序从入口指令开始，逐条执行程序中的指令。

5. **进程结束**  
   执行完所有指令后，进程终止，释放分配的资源。

## C语言入口函数定义

C 语言的入口函数是 `main` 函数，其定义如下：

argc: 操作系统传递给程序的参数个数。

argv: 操作系统传给我们的一个字符串数组，包含所有传递给程序的参数。argv 是 char* 类型的数组，其定义为 char**。

```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main(int argc, char** argv) {
    printf("argc = %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("argv[%d] = %s\n", i, argv[i]);
    }
    system("pause");
    return 0;
}
