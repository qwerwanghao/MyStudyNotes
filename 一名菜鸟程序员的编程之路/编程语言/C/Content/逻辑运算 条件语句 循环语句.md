# 学习笔记： 逻辑运算 条件语句 循环语句

## 一、顺序执行与函数调用

1. **顺序执行**: 
   - 一条一条语句依次执行，直到程序结束。

2. **函数调用**:
   - 调用函数时，程序进入函数体内部执行代码。
   - 当遇到 `return` 语句时，程序返回到调用该函数的位置，继续往下执行。
   - 可以使用调试器查看整个执行过程，以便理解程序的运行逻辑。

---

## 二、逻辑运算

1. **真与假**:
   - 非零值表示为真，零值表示为假。
   - 逻辑判断成立为真，不成立为假。
   - 常用逻辑比较运算符：`>`, `>=`, `<`, `<=`, `==`, `!=`。

2. **逻辑运算符**:
   - **逻辑与 (`&&`)**:
     - 真 && 真 --> 真  
     - 假 && 假 --> 假  
     - 真 && 假 --> 假  
     - 假 && 真 --> 假  

   - **逻辑或 (`||`)**:
     - 真 || 真 --> 真  
     - 假 || 假 --> 假  
     - 真 || 假 --> 真  
     - 假 || 真 --> 真  

---

## 三、条件判断语句

1. **`if` 语句**:
   - 基本语法:
     ```c
     if (条件) {
         // 条件为真时执行的代码
     }
     ```

2. **`if-else` 语句**:
   - 基本语法:
     ```c
     if (条件) {
         // 条件为真时执行的代码
     } else {
         // 条件为假时执行的代码
     }
     ```

3. **`if-else if-else` 语句**:
   - 基本语法:
     ```c
     if (条件1) {
         // 条件1为真时执行的代码
     } else if (条件2) {
         // 条件2为真时执行的代码
     } else {
         // 所有条件均不满足时执行的代码
     }
     ```

4. **`switch` 语句**:
   - 基本语法:
     ```c
     switch (值) {
         case 值1:
             // 执行的代码
             break;
         case 值2:
             // 执行的代码
             break;
         default:
             // 执行的代码
             break;
     }
     ```
