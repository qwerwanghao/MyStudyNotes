# C语言函数详解

## 函数基础

### 函数声明与定义
```c
// 函数原型（声明）
返回类型 函数名(参数列表);

// 函数定义
返回类型 函数名(参数列表) {
    // 函数体
    return 返回值;
}
```

### 参数传递方式
```c
// 值传递
void swap_wrong(int a, int b) {  // 不会真正交换
    int temp = a;
    a = b;
    b = temp;
}

// 指针传递
void swap_correct(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

// 数组参数
void processArray(int arr[], int size);  // arr退化为指针
void processArray(int *arr, int size);   // 等价形式
```

## 函数高级特性

### 内联函数
```c
inline int max(int a, int b) {
    return (a > b) ? a : b;
}
```

### 可变参数
```c
#include <stdarg.h>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    
    int result = 0;
    for(int i = 0; i < count; i++) {
        result += va_arg(args, int);
    }
    
    va_end(args);
    return result;
}
```

### 函数指针
```c
// 定义函数指针类型
typedef int (*Operation)(int, int);

// 使用函数指针
int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }

Operation op = add;
int result = op(5, 3);  // 调用add(5, 3)
```

## 递归函数

### 基本模式
```c
int factorial(int n) {
    if (n <= 1) return 1;  // 基准情况
    return n * factorial(n - 1);  // 递归情况
}
```

### 尾递归优化
```c
// 普通递归
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}

// 尾递归优化
int fibTail(int n, int a, int b) {
    if (n == 0) return a;
    return fibTail(n-1, b, a+b);
}
```

## 作用域与生命周期

### 静态函数
```c
static int helper(int x) {  // 仅在当前文件可见
    return x * x;
}
```

### 外部函数
```c
// 在头文件中
extern int publicFunction(int x);

// 在源文件中
int publicFunction(int x) {
    return x + 1;
}
```

## 错误处理

### 返回错误码
```c
int divide(int a, int b, int *result) {
    if (b == 0) {
        return -1;  // 错误码
    }
    *result = a / b;
    return 0;      // 成功
}
```

### 使用errno
```c
#include <errno.h>
#include <string.h>

void handleError() {
    fprintf(stderr, "Error: %s\n", strerror(errno));
}
```

## 最佳实践

### 1. 函数设计原则
```c
// 单一职责
void processData(const Data *data) {
    validateData(data);     // 验证
    transformData(data);    // 转换
    saveData(data);        // 保存
}
```

### 2. 参数验证
```c
int *createArray(size_t size) {
    if (size == 0) {
        return NULL;
    }
    return malloc(size * sizeof(int));
}
```

### 3. 资源管理
```c
FILE *openFile(const char *filename) {
    FILE *file = fopen(filename, "r");
    if (!file) {
        handleError();
        return NULL;
    }
    return file;
}
```
