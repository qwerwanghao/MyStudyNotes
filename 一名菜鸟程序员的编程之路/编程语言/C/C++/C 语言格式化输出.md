# 学习笔记：C 语言格式化输出

---

## 格式说明符汇总
| 类型            | 格式说明符 | 描述                                     | 示例输出          |
|-----------------|------------|------------------------------------------|-------------------|
| 有符号整数      | `%d`, `%i` | 有符号十进制整数                         | `42`              |
| 无符号整数      | `%u`       | 无符号十进制整数                         | `42`              |
| 八进制整数      | `%o`       | 八进制表示                               | `52`              |
| 十六进制整数    | `%x`, `%X` | 十六进制表示（小写/大写字母）            | `2a` / `2A`        |
| 浮点数          | `%f`       | 标准浮点数表示                           | `3.141590`        |
| 科学计数法      | `%e`, `%E` | 科学计数法表示（小写/大写字母）          | `3.141590e+00` / `3.141590E+00` |
| 最短表示        | `%g`       | 根据值选择`%e`或`%f`中较短的表示          | `3.14159`         |
| 字符            | `%c`       | 单个字符                                 | `A`               |
| 字符串          | `%s`       | 字符串                                   | `Hello, World!`   |
| 指针地址        | `%p`       | 内存地址（需转换为`void*`）              | `0x7ffee3bff5ac`  |
| 宽度控制        | `%10d`     | 最小宽度为10，右对齐                     | `        42`      |
| 左对齐          | `%-10d`    | 最小宽度为10，左对齐                     | `42        end`    |
| 精度控制        | `%.2f`     | 浮点数保留2位小数                        | `3.14`            |
| 最小宽度与精度  | `%6.3f`    | 宽度6，小数点后3位                        | ` 3.142`          |
| 前导零          | `%05.2f`   | 宽度5，小数点后2位，前导零填充            | `03.14`           |
| 百分号          | `%%`       | 输出百分号`%`                             | `%`               |
