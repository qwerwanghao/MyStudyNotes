# 流程控制

## 条件语句

### if语句
1. **基本if语句**
   ```csharp
   if (condition)
   {
       // 条件为true时执行的代码
   }
   ```

2. **if-else语句**
   ```csharp
   if (score >= 60)
   {
       Console.WriteLine("通过");
   }
   else
   {
       Console.WriteLine("未通过");
   }
   ```

3. **if-else if-else语句**
   ```csharp
   if (score >= 90)
   {
       grade = 'A';
   }
   else if (score >= 80)
   {
       grade = 'B';
   }
   else if (score >= 70)
   {
       grade = 'C';
   }
   else
   {
       grade = 'D';
   }
   ```

### switch语句
1. **基本switch语句**
   ```csharp
   switch (gameState)
   {
       case GameState.Menu:
           ShowMenu();
           break;
       case GameState.Playing:
           UpdateGame();
           break;
       case GameState.Paused:
           HandlePause();
           break;
       default:
           HandleUnknownState();
           break;
   }
   ```

2. **模式匹配switch**
   ```csharp
   switch (obj)
   {
       case int i:
           Console.WriteLine($"整数：{i}");
           break;
       case string s:
           Console.WriteLine($"字符串：{s}");
           break;
       case Player p when p.IsAlive:
           p.Update();
           break;
       default:
           Console.WriteLine("未知类型");
           break;
   }
   ```

## 循环语句

### for循环
1. **基本for循环**
   ```csharp
   for (int i = 0; i < 10; i++)
   {
       Console.WriteLine(i);
   }
   ```

2. **嵌套for循环**
   ```csharp
   for (int i = 0; i < 3; i++)
   {
       for (int j = 0; j < 3; j++)
       {
           Console.WriteLine($"({i},{j})");
       }
   }
   ```

### while循环
1. **while循环**
   ```csharp
   int health = 100;
   while (health > 0)
   {
       UpdateGame();
       health -= damage;
   }
   ```

2. **do-while循环**
   ```csharp
   string input;
   do
   {
       input = Console.ReadLine();
       ProcessInput(input);
   } while (input != "quit");
   ```

### foreach循环
1. **数组遍历**
   ```csharp
   int[] numbers = { 1, 2, 3, 4, 5 };
   foreach (int num in numbers)
   {
       Console.WriteLine(num);
   }
   ```

2. **集合遍历**
   ```csharp
   List<Enemy> enemies = GetEnemies();
   foreach (Enemy enemy in enemies)
   {
       enemy.Update();
   }
   ```

## 跳转语句

### break语句
```csharp
// 跳出循环
for (int i = 0; i < 100; i++)
{
    if (IsGameOver())
    {
        break;
    }
    UpdateGame();
}

// 跳出switch
switch (input)
{
    case "quit":
        ExitGame();
        break;
}
```

### continue语句
```csharp
// 跳过当前迭代
for (int i = 0; i < enemies.Length; i++)
{
    if (!enemies[i].isActive)
    {
        continue;
    }
    enemies[i].Update();
}
```

### return语句
```csharp
public bool CheckCollision(GameObject obj)
{
    if (obj == null)
    {
        return false;
    }
    
    // 检查碰撞
    return IsColliding(obj);
}
```

## Unity特殊流程控制

### 协程
1. **基本协程**
   ```csharp
   IEnumerator SpawnEnemies()
   {
       while (true)
       {
           SpawnEnemy();
           yield return new WaitForSeconds(2f);
       }
   }
   ```

2. **条件等待**
   ```csharp
   IEnumerator WaitForAnimation()
   {
       animator.Play("Attack");
       yield return new WaitForSeconds(attackDuration);
       ResetState();
   }
   ```

### 事件系统
1. **Unity事件**
   ```csharp
   void OnEnable()
   {
       playerHealth.onDeath += GameOver;
   }

   void OnDisable()
   {
       playerHealth.onDeath -= GameOver;
   }
   ```

2. **自定义事件**
   ```csharp
   public event System.Action<int> onScoreChange;

   void UpdateScore(int newScore)
   {
       score = newScore;
       onScoreChange?.Invoke(score);
   }
   ```

## 异常处理

### try-catch语句
```csharp
try
{
    LoadGameData();
}
catch (FileNotFoundException e)
{
    Debug.LogError("找不到存档文件: " + e.Message);
    CreateNewGame();
}
catch (Exception e)
{
    Debug.LogError("加载游戏出错: " + e.Message);
}
finally
{
    CleanupResources();
}
```

### 异常抛出
```csharp
public void SetHealth(int value)
{
    if (value < 0)
    {
        throw new ArgumentException("生命值不能为负数");
    }
    health = value;
}
