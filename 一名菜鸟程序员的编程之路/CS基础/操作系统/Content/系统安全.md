# 系统安全

## 访问控制

### 访问控制矩阵
1. **基本概念**
   - 主体（用户/进程）
   - 客体（文件/资源）
   - 访问权限
   - 访问规则

2. **实现方式**
   - 访问控制表（ACL）
   - 能力表（Capabilities）
   - 角色访问控制
   - 强制访问控制

### 访问控制策略
1. **自主访问控制（DAC）**
   - 所有者控制
   - 权限委托
   - 灵活性高
   - 管理复杂

2. **强制访问控制（MAC）**
   - 系统强制执行
   - 安全级别
   - 信息流控制
   - 军事应用

3. **基于角色的访问控制（RBAC）**
   - 角色定义
   - 权限分配
   - 角色继承
   - 职责分离

3. **令牌认证**

   - 增强安全性

2. **三因素认证**
   - 知识因素

## 安全机制

### 防火墙
1. **类型**
   - 包过滤
   - 应用网关
   - 状态检测
   - 代理服务器

2. **功能**
   - 访问控制
   - 流量监控
   - 日志记录
   - 入侵检测

### 加密系统
1. **对称加密**
   - 共享密钥
   - 加解密速度快
   - 密钥管理困难
   - DES/AES算法

2. **非对称加密**
   - 公钥私钥
   - 安全性高
   - 计算开销大
   - RSA算法

3. **混合加密**
   - 结合两种方式
   - 会话密钥
   - 数字信封
   - 实用性强

### 入侵检测
1. **检测方式**
   - 特征检测
   - 异常检测
   - 状态监控
   - 行为分析

2. **响应措施**
   - 报警通知
   - 阻断连接
   - 日志记录
   - 系统恢复
