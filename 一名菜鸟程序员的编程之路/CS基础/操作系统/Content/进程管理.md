# 进程管理

## 进程概念

### 进程定义
1. **基本概念**
   - 程序的执行实例
   - 动态的过程
   - 资源分配的基本单位
   - 独立的地址空间

2. **进程特征**
   - 动态性
   - 并发性
   - 独立性
   - 异步性

### 进程控制块（PCB）
1. **基本信息**
   - 进程标识符
   - 进程状态
   - 程序计数器
   - 进程优先级

2. **资源信息**
   - CPU寄存器
   - 内存管理信息
   - 文件管理信息
   - I/O设备信息

## 进程状态与转换

### 进程状态
1. **创建状态**
   - 分配PCB
   - 初始化进程
   - 加入就绪队列

2. **就绪状态**
   - 具备运行条件
   - 等待CPU分配
   - 进程调度对象

3. **运行状态**
   - 占用CPU
   - 执行指令
   - 时间片控制

4. **阻塞状态**
   - 等待事件
   - 无法继续执行
   - 释放CPU

5. **终止状态**
   - 执行完毕
   - 异常终止
   - 资源回收

### 状态转换
1. **创建→就绪**
   - 系统资源满足
   - 初始化完成

2. **就绪→运行**
   - 被调度程序选中
   - 分配CPU资源

3. **运行→就绪**
   - 时间片用完
   - 高优先级进程到达

4. **运行→阻塞**
   - 等待I/O完成
   - 等待资源分配
   - 等待信号

5. **阻塞→就绪**
   - 等待事件完成
   - 资源获得满足

## 进程调度

### 调度层次
1. **高级调度（作业调度）**
   - 调入磁盘作业
   - 创建新进程
   - 长期调度

2. **中级调度（内存调度）**
   - 内存空间管理
   - 进程挂起和激活
   - 中期调度

3. **低级调度（进程调度）**
   - CPU分配
   - 进程切换
   - 短期调度

### 调度算法
1. **先来先服务（FCFS）**
   - 按到达顺序
   - 公平简单
   - 平均等待时间可能较长

2. **短作业优先（SJF）**
   - 最短执行时间优先
   - 最小平均等待时间
   - 可能产生饥饿

3. **优先级调度**
   - 静态优先级
   - 动态优先级
   - 优先级反转问题

4. **时间片轮转**
   - 公平分配CPU
   - 响应时间短
   - 上下文切换开销大

5. **多级反馈队列**
   - 多个优先级队列
   - 动态调整优先级
   - 兼顾公平和效率

## 死锁处理

### 死锁条件
1. **互斥条件**
   - 资源独占
   - 不可共享

2. **请求和保持**
   - 持有资源
   - 请求新资源

3. **不可抢占**
   - 资源不可强制释放
   - 只能主动放弃

4. **循环等待**
   - 循环依赖链
   - 资源请求环

### 死锁处理策略
1. **死锁预防**
   - 破坏死锁条件
   - 静态分配
   - 资源有序分配

2. **死锁避免**
   - 银行家算法
   - 安全状态
   - 动态分配

3. **死锁检测**
   - 资源分配图
   - 死锁检测算法
   - 周期性检测

4. **死锁恢复**
   - 进程终止
   - 资源抢占
   - 回滚恢复

## 线程管理

### 线程概念
1. **定义**
   - 轻量级进程
   - 执行的基本单位
   - 共享进程资源

2. **特点**
   - 资源共享
   - 并发执行
   - 上下文切换快

### 线程类型
1. **用户级线程**
   - 用户空间管理
   - 系统不可见
   - 切换开销小

2. **内核级线程**
   - 系统直接支持
   - 调度灵活
   - 开销较大

3. **混合型线程**
   - 多对多模型
   - 灵活性好
   - 复杂度高

### 线程同步
1. **互斥锁**
   - 基本同步机制
   - 原子操作
   - 死锁风险

2. **条件变量**
   - 等待/通知机制
   - 线程协作
   - 避免忙等
