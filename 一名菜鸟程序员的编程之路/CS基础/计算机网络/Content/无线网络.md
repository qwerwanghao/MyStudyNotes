# 无线网络

## WiFi

### IEEE 802.11标准
1. **802.11b**
   - 2.4GHz频段
   - 最高11Mbps
   - 向后兼容
   - 覆盖范围大

2. **802.11g**
   - 2.4GHz频段
   - 最高54Mbps
   - 兼容802.11b
   - 抗干扰性improved

3. **802.11n**
   - 2.4GHz和5GHz双频段
   - MIMO技术
   - 最高600Mbps
   - 更好的覆盖范围

4. **802.11ac**
   - 5GHz频段
   - MU-MIMO
   - 最高6.9Gbps
   - 波束成形技术

### WiFi工作原理
1. **基本架构**
   - 接入点（AP）
   - 无线客户端
   - 分布式系统
   - 基本服务集（BSS）

2. **信道管理**
   - 信道选择
   - 信道干扰
   - 动态频率选择
   - 发射功率控制

3. **安全机制**
   - WEP加密
   - WPA/WPA2加密
   - WPA3加密
   - MAC地址过滤

## 蓝牙

### 蓝牙技术特点
1. **技术规格**
   - 2.4GHz频段
   - 跳频扩频
   - 短距离通信
   - 低功耗

2. **工作模式**
   - 传统蓝牙
   - 低功耗蓝牙（BLE）
   - 双模蓝牙

3. **应用场景**
   - 无线音频
   - 数据传输
   - 物联网设备
   - 近场通信

### 蓝牙协议栈
1. **底层协议**
   - 基带层
   - 链路管理层
   - 主机控制接口

2. **上层协议**
   - L2CAP
   - RFCOMM
   - SDP
   - 应用层协议

## 5G技术

### 5G特点
1. **性能指标**
   - 超高速率（20Gbps）
   - 超低时延（1ms）
   - 大连接（1M/km²）
   - 高可靠性

2. **关键技术**
   - 大规模MIMO
   - 毫米波通信
   - 网络切片
   - 边缘计算

3. **应用场景**
   - 增强移动宽带（eMBB）
   - 超高可靠低时延通信（uRLLC）
   - 大规模机器通信（mMTC）

### 5G网络架构
1. **核心网**
   - 服务化架构
   - 网络功能虚拟化
   - 软件定义网络
   - 云化部署

2. **接入网**
   - 宏基站
   - 小基站
   - 分布式天线
   - 回传网络
